жөҒеӨ„зҗҶеҝғжҷә:ж—¶й—ҙгҖҒзӘ—еҸЈгҖҒзҠ¶жҖҒгҖҒExactly-Once вҖӢ

жү№еӨ„зҗҶзҡ„дё–з•ҢеҫҲе®үйқҷ:ж•°жҚ®жҳҜгҖҢиҝҮеҺ»гҖҚгҖҒжҳҜгҖҢе…ЁйғЁгҖҚгҖҒи·‘е®Ңе°ұз»“жқҹгҖӮжөҒеӨ„зҗҶзҡ„дё–з•ҢеҫҲд№ұ:ж•°жҚ®жҳҜгҖҢжӯЈеңЁеҲ°иҫҫзҡ„гҖҚгҖҒжҳҜгҖҢж°ёиҝңжІЎе®ҢгҖҚгҖҒиҝҳеҸҜиғҪд№ұеәҸеҲ°иҫҫ(жүӢжңәзҰ»зәҝ 3 е°Ҹж—¶еҗҺдёҖиө·дёҠдј )гҖӮжүҖд»ҘжөҒеӨ„зҗҶеҮ д№ҺйҮҚж–°е®ҡд№үдәҶгҖҢж—¶й—ҙгҖҚгҖҢзӘ—еҸЈгҖҚгҖҢзҠ¶жҖҒгҖҚгҖҢжӯЈзЎ®жҖ§гҖҚиҝҷеӣӣд»¶дәӢвҖ”вҖ”жү№зҡ„зӣҙи§үиҝҮжқҘе…ЁжҳҜеқ‘гҖӮиҝҷдёҖзҜҮжӢүжё…жҘҡжөҒеӨ„зҗҶзҡ„еӣӣд»¶еә•еӯҗ,зҗҶи§ЈдәҶиҝҷдёҖзҜҮ,19/20/21 зҜҮзҡ„ Watermark / Flink / зҠ¶жҖҒеҗҺз«Ҝ жүҚжңүж„Ҹд№үгҖӮ

дёҖеҸҘиҜқе…Ҳи®°дҪҸ:жөҒеӨ„зҗҶ = ж—¶й—ҙ + зӘ—еҸЈ + зҠ¶жҖҒ + е®№й”ҷиҜӯд№үгҖӮдәӢд»¶ж—¶й—ҙ vs еӨ„зҗҶж—¶й—ҙжҳҜжөҒеӨ„зҗҶзҡ„з¬¬дёҖйҒ“еқҺ,зӘ—еҸЈеҶіе®ҡгҖҢд»Җд№Ҳж—¶еҖҷеҮәз»“жһңгҖҚ,зҠ¶жҖҒеҶіе®ҡгҖҢи®°дҪҸд»Җд№ҲгҖҚ,Exactly-Once дёҚжҳҜгҖҢж•°жҚ®еҸӘеӨ„зҗҶдёҖж¬ЎгҖҚ,жҳҜ**гҖҢз»“жһңеҜ№еӨ–еҸҜи§ҒеҸӘеҸ‘з”ҹдёҖж¬ЎгҖҚ**(з«ҜеҲ°з«ҜдәӢеҠЎ)гҖӮ

дёүгҖҒзӘ—еҸЈ:д»Җд№Ҳж—¶еҖҷеҮәз»“жһң вҖӢ

3.1 жөҒеӨ„зҗҶйңҖиҰҒзӘ—еҸЈзҡ„еҺҹеӣ  вҖӢ

жөҒжҳҜж— з•Ңзҡ„,иҒҡеҗҲ(SUM / COUNT / AVG)жІЎжңүгҖҢе…ЁйғЁгҖҚжҰӮеҝөгҖӮжүҖд»Ҙеҝ…йЎ»еҲҮзӘ—еҸЈ:"жҜҸ 5 еҲҶй’ҹ,жҠҠиҝҷ 5 еҲҶй’ҹзҡ„ж•°жҚ®иҒҡеҗҲжҲҗдёҖдёӘиҫ“еҮә"гҖӮ

3.2 дёүз§Қдё»жөҒзӘ—еҸЈ вҖӢ

1. еӣәе®ҡзӘ—еҸЈ (Tumbling Window)
   |---5 min---|---5 min---|---5 min---|
   дёҚйҮҚеҸ ,жҜҸжқЎдәӢд»¶еұһдәҺдёҖдёӘзӘ—еҸЈ
   
   дҫӢ:жҜҸ 5 еҲҶй’ҹз»ҹи®Ў GMV
   12:00-12:05 вҶ’ 30дёҮ
   12:05-12:10 вҶ’ 28дёҮ
   ...

2. ж»‘еҠЁзӘ—еҸЈ (Sliding Window)
   |---5 min---|
        |---5 min---|
             |---5 min---|
   зӘ—еҸЈеӨ§е°Ҹ 5 min,жҜҸ 1 min ж»‘еҠЁ
   жҜҸжқЎдәӢд»¶еҸҜиғҪеұһдәҺеӨҡдёӘзӘ—еҸЈ
   
   дҫӢ:жңҖиҝ‘ 5 еҲҶй’ҹ GMV,жҜҸ 1 еҲҶй’ҹжӣҙж–°

3. дјҡиҜқзӘ—еҸЈ (Session Window)
   |--user A жҙ»и·ғ--|        |--user A еҸҲжҙ»и·ғ--|
                  вҶ‘
              5 min дёҚжҙ»и·ғ вҶ’ еҲҮзӘ—еҸЈ
   жҢүз”ЁжҲ·жҙ»еҠЁй—ҙйҡ”еҲҮ
   
   дҫӢ:з”ЁжҲ·иҝһз»ӯжөҸи§ҲдҪңдёәдёҖдёӘ session,gap 5 min еҲҮ

3.3 и§ҰеҸ‘(Trigger) вҖӢ

гҖҢиҝҷдёӘзӘ—еҸЈд»Җд№Ҳж—¶еҖҷз®—е®Ң,еҮәз»“жһң?гҖҚ

Event Time Trigger:  Watermark и¶ҠиҝҮ window end ж—¶(19 зҜҮеұ•ејҖ)
Processing Time Trigger: жҢүзү©зҗҶж—¶й’ҹеҲ°зӮ№
Count Trigger:        зҙҜи®Ў N дёӘдәӢд»¶еҗҺ
Composite Trigger:    з»„еҗҲ(ж—©еҮәиҝ‘дјј + жңҖз»ҲзІҫзЎ®)

3.4 е·ҘзЁӢдҫӢеӯҗ(Flink SQL) вҖӢ

sql
-- еӣәе®ҡзӘ—еҸЈ
SELECT
    TUMBLE_START(event_time, INTERVAL '5' MINUTE) AS win_start,
    SUM(amount) AS gmv
FROM orders
GROUP BY TUMBLE(event_time, INTERVAL '5' MINUTE);

-- ж»‘еҠЁзӘ—еҸЈ
SELECT
    HOP_START(event_time, INTERVAL '1' MINUTE, INTERVAL '5' MINUTE) AS win_start,
    SUM(amount) AS gmv
FROM orders
GROUP BY HOP(event_time, INTERVAL '1' MINUTE, INTERVAL '5' MINUTE);

-- дјҡиҜқзӘ—еҸЈ
SELECT
    SESSION_START(event_time, INTERVAL '5' MINUTE) AS win_start,
    user_id,
    COUNT(*) AS clicks
FROM clicks
GROUP BY SESSION(event_time, INTERVAL '5' MINUTE), user_id;

жЁЎејҸеҢ№й…Қ CEP:           дҝқз•ҷдәӢд»¶еәҸеҲ— = зҠ¶жҖҒ
жңҖиҝ‘ 5 еҲҶй’ҹз»ҹи®Ў:        зӘ—еҸЈеҶ…зҡ„жүҖжңүдәӢд»¶ = зҠ¶жҖҒ

дә”гҖҒдәӨд»ҳиҜӯд№ү:At Most / At Least / Exactly Once вҖӢ

5.1 дёүз§ҚиҜӯд№ү вҖӢ

At Most Once     жңҖеӨҡдёҖж¬Ў
                 еҸҜиғҪдёўж¶ҲжҒҜ,дёҚйҮҚеӨҚ
                 жІЎдәәеңЁз”ҹдә§з”Ё(йҷӨйқһеҸҜдёўж•°жҚ®зҡ„зӣ‘жҺ§)

At Least Once    иҮіе°‘дёҖж¬Ў
                 дёҚдёўж¶ҲжҒҜ,еҸҜиғҪйҮҚеӨҚ

-- Flink SQL
CREATE TABLE kafka_orders (
    id BIGINT,
    user_id BIGINT,
    province STRING,
    amount DECIMAL(10,2),
    event_time TIMESTAMP(3),
    WATERMARK FOR event_time AS event_time - INTERVAL '1' MINUTE
) WITH (
    'connector' = 'kafka',
    'topic' = 'orders',
    'properties.bootstrap.servers' = 'kafka:9092',
    'format' = 'avro-confluent',
    'avro-confluent.url' = 'http://schema-registry:8081'
);

CREATE TABLE clickhouse_gmv (
    win_start TIMESTAMP(3),
    province STRING,
    gmv DECIMAL(20,2),
    PRIMARY KEY (win_start, province) NOT ENFORCED
) WITH ('connector' = 'clickhouse', ...);

INSERT INTO clickhouse_gmv
SELECT
    TUMBLE_START(event_time, INTERVAL '5' MINUTE) AS win_start,
    province,
    SUM(amount) AS gmv
FROM kafka_orders
GROUP BY 
    TUMBLE(event_time, INTERVAL '5' MINUTE),
    province;

# 2D 光影、法线贴图与伪 3D 质感

2D 光影不是让游戏变“真实”,而是让平面图看起来有厚度、有时间感、有重点。火把、爆炸、暗室、受击闪光,都可以从这套系统开始。

> 一句话先记住:**光源越多越贵,阴影更贵;先设计光照层,再往场景里加灯。**

---

## 一、最小 2D 光照

场景里加:

```text
PointLight2D
```

给它一张径向渐变贴图,调:

```text
color
energy
texture_scale
height
```

让某个 Sprite 接受光,它的 `light_mask` 要和光源的 cull mask 对上。看不到光时,先查 mask,别先怀疑引擎。

---

## 二、光不是免费的

一盏没阴影的灯通常还能接受。几十盏动态灯,加上阴影,就会明显吃性能。

简单预算:

```text
常亮环境光: 1 个 DirectionalLight2D 或 CanvasModulate
火把/灯笼: 屏幕内 5-12 个
爆炸/技能: 短时间临时光,播完就关
带阴影光源: 少用,优先给重要光源
```

不要给每颗子弹都挂永久 `PointLight2D`。需要拖尾发光时,很多时候 shader 或粒子更便宜。

---

## 三、CanvasModulate 做全局氛围

暗场景可以先放一个:

```text
CanvasModulate
```

把整体颜色压暗:

```text
color = 深蓝灰
```

然后用 `PointLight2D` 打局部亮光。这样比给所有东西单独调 `modulate` 干净。

---

## 四、法线贴图怎么用

普通 sprite 是平的。法线贴图告诉光照系统“这个像素朝哪个方向”。

Godot 2D 通常用 `CanvasTexture`:

```text
CanvasTexture
  diffuse_texture = 原图
  normal_texture  = 法线图
```

然后把 `Sprite2D.texture` 换成这个 `CanvasTexture`。

如果法线没效果,常见原因:

- 没有实际光源。
- 光源 `height` 太低。
- normal map 的 Y 方向反了。
- Sprite 没有接收这盏光的 mask。

`PointLight2D.height` 可以先试 64 或 128。不是越高越好,太高会变成一片平光。

---

## 五、阴影用 LightOccluder2D

挡光不是用碰撞体,而是:

```text
LightOccluder2D
└── OccluderPolygon2D
```

给墙、柱子、大石块画遮挡多边形。开启光源:

```text
shadow_enabled = true
```

注意:

- 阴影只对开启 shadow 的光生效。
- occluder mask 要和光源匹配。
- 遮挡形状不要太复杂。
- 小装饰物不一定需要投影。

碰撞多边形和遮挡多边形可以长得像,但职责不同。不要指望 `CollisionPolygon2D` 自动挡光。

---

## 六、light_mask 要规划

可以给光照也做层:

```text
1 world_light      地图和角色接收
2 effect_light     技能特效接收
3 ui_light         少用,通常 UI 不接世界光
```

角色、地面、装饰的 `light_mask` 要一致。HUD 放 `CanvasLayer`,通常不受世界光影响。

如果某个物体“不亮”,检查三件事:

```text
Sprite2D.light_mask
Light2D.range_item_cull_mask
Light2D.enabled
```

## 七、白天黑夜先别做复杂

先用一个 `CanvasModulate` 过渡:

```gdscript
@export var day_color: Color = Color.WHITE
@export var night_color: Color = Color(0.18, 0.2, 0.35)

@onready var _modulate: CanvasModulate = %CanvasModulate

func set_night_amount(t: float) -> void:
    _modulate.color = day_color.lerp(night_color, clampf(t, 0.0, 1.0))
```

再决定哪些灯在夜晚打开:

```gdscript
func set_lamps_enabled(enabled: bool) -> void:
    for lamp in get_tree().get_nodes_in_group(&"lamp"):
        lamp.enabled = enabled
```

---

## 八、验收

- 场景有全局氛围光,不是全靠几十盏小灯。
- 重要灯光才开阴影。
- 法线贴图通过 `CanvasTexture` 接入。
- light mask 有命名和约定。
- HUD 不被世界光影响。
- 灯光数量有预算。

---

## 常见坑

**坑 1:看不到光就乱调 energy。**

先查 mask 和层级。

**坑 2:每个装饰都投阴影。**

阴影贵。只给会影响视觉判断的大物体投。

**坑 3:法线贴图挂错位置。**

Sprite 要用 `CanvasTexture`,不是只导入一张 normal 图就会自动生效。

**坑 4:用光源做所有发光特效。**

短时技能光可以,大量发光子弹更适合粒子或 shader。

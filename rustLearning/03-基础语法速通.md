# 03-基础语法速通

> "Rust 的语法是 C 的大括号风格混血了函数式编程的灵魂。"

既然你已经有其他语言的基础，这篇我们只看 Rust 最特殊或最核心的语法差异。

## 一、变量绑定与可变性

在 Rust 中，变量默认是**不可变（immutable）**的。这也是安全性的基石。

```rust
fn main() {
    let x = 5;
    // x = 6; // ❌ 报错：不能给不可变变量重新赋值

    let mut y = 5; // 必须加 mut 才能修改
    y = 6; // ✅ 正确
}
```

### 常量 vs 不可变变量
- 常量用 `const` 声明，且必须显式标注类型。
- 常量不仅默认不可变，而且永远不能是 `mut`。
- 变量的值在运行时计算，常量的值必须在**编译时**就能确定。

```rust
const MAX_POINTS: u32 = 100_000;
```

## 二、数据类型

Rust 是静态强类型语言。

1. **标量类型**：
   - 整数：`i32` (默认), `u32`, `i64`, `u8` (常用作字节) 等。
   - 浮点数：`f64` (默认), `f32`。
   - 布尔：`bool`。
   - 字符：`char` (Rust 的 char 占 4 个字节，是 Unicode 标量值，支持 emoji 🦁)。

2. **复合类型**：
   - **元组 (Tuple)**：长度固定，类型可以不同。
     ```rust
     let tup: (i32, f64, u8) = (500, 6.4, 1);
     let (x, y, z) = tup; // 解构
     let five_hundred = tup.0; // 点号访问
     ```
   - **数组 (Array)**：长度固定，类型必须相同。分配在栈上。
     ```rust
     let a = [1, 2, 3, 4, 5];
     let b: [i32; 5] = [1, 2, 3, 4, 5];
     let c = [3; 5]; // [3, 3, 3, 3, 3]
     ```
   *注意：如果你想要长度可变的数组，你需要的是 `Vector`（后续讲）。*

## 三、函数与隐式返回

Rust 函数名使用 `snake_case`（下划线命名法）。

```rust
fn add_numbers(x: i32, y: i32) -> i32 {
    let sum = x + y;
    sum // 👈 注意：这里没有 return，也没有分号！
}
```
**核心概念：语句（Statement）与表达式（Expression）**
- **语句**执行操作但不返回值（以分号结尾）。`let x = 5;` 是语句。
- **表达式**计算并产生一个值。
- 函数的最后一行如果**不加分号**，它就是一个表达式，其计算结果会自动作为函数的返回值。这被称为**尾表达式**。
- 当然，你也可以用 `return sum;` 提前返回，但在函数末尾使用尾表达式是 Rust 的地道写法 (Idiomatic Rust)。

## 四、控制流

### 1. if 表达式
因为 `if` 是表达式，它可以直接用来赋值（类似三元运算符）：
```rust
let condition = true;
let number = if condition { 5 } else { 6 };
```

### 2. 循环 (loop, while, for)
- `loop`：无限循环。可以返回值。
  ```rust
  let mut counter = 0;
  let result = loop {
      counter += 1;
      if counter == 10 {
          break counter * 2; // break 后跟要返回的值
      }
  };
  ```
- `while`：当条件为真时执行。
- `for`：遍历集合的最佳选择。
  ```rust
  let a = [10, 20, 30, 40, 50];
  for element in a {
      println!("the value is: {}", element);
  }

  // 遍历数字范围 (左闭右开)
  for number in 1..4 {
      println!("{}!", number);
  }
  ```

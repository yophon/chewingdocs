гҖҢжңҖеҘҪзҡ„йҳІеҫЎе·ҘзЁӢеёҲеҝ…йЎ»жҮӮж”»еҮ»гҖҚвҖ”вҖ”дҪҶжҮӮж”»еҮ» вү  е®һж–Ҫж”»еҮ»гҖӮиҝҷдёҖз« и®Ізҡ„жҳҜжё—йҖҸжөӢиҜ•зҡ„ж–№жі•и®әе’Ңе·Ҙе…·й“ҫ,зӣ®ж ҮжҳҜи®©еҒҡйҳІеҫЎзҡ„е·ҘзЁӢеёҲзҗҶи§Јж”»еҮ»иҖ…жҖҺд№ҲжҖқиҖғ,д»ҺиҖҢиғҪеҶҷеҮәжӣҙжңүй’ҲеҜ№жҖ§зҡ„ WAF и§„еҲҷгҖҒйҷҗжөҒзӯ–з•Ҙе’Ңзӣ‘жҺ§жҢҮж ҮгҖӮ

дёҖеҸҘиҜқе…Ҳи®°дҪҸ:жё—йҖҸжөӢиҜ•жҳҜжңүжі•еҫӢиҫ№з•Ңзҡ„е·ҘзЁӢиЎҢдёәвҖ”вҖ”жҺҲжқғиҢғеӣҙеҶ…гҖҒзӯҫзҪІ SOW(е·ҘдҪңиҜҙжҳҺд№Ұ)/ROE(дәӨжҲҳи§„еҲҷ)гҖҒзәҰе®ҡжөӢиҜ•ж—¶й—ҙзӘ—еҸЈгҖҒйҷҗе®ҡдёҚеҪұе“Қз”ҹдә§ вҶ’ иҝҷжүҚжҳҜжё—йҖҸгҖӮжІЎжҺҲжқғжү«з«ҜеҸЈйғҪжҳҜиҝқжі•зҡ„:дёӯеӣҪгҖҠеҲ‘жі•гҖӢ285 жқЎ(йқһжі•дҫөе…Ҙи®Ўз®—жңәдҝЎжҒҜзі»з»ҹзҪӘ)/286 жқЎ(з ҙеқҸи®Ўз®—жңәдҝЎжҒҜзі»з»ҹзҪӘ)/253 жқЎд№ӢдёҖ(дҫөзҠҜе…¬ж°‘дёӘдәәдҝЎжҒҜзҪӘ),зҫҺеӣҪ CFAAгҖҒж¬§зӣҹ NIS2 йғҪжҳҺзЎ®е…ҘзҪӘгҖӮжң¬з« еҸӘи®ІйҳІеҫЎи§Ҷи§’:ж”»еҮ»е·Ҙе…·й•ҝд»Җд№Ҳж ·гҖҒе…ҘеҸЈеңЁе“ӘйҮҢгҖҒжҖҺд№Ҳд»Һж”»еҮ»ж—Ҙеҝ—еҸҚжҺЁйҳІеҫЎзӯ–з•ҘвҖ”вҖ”жүҖжңүе‘Ҫд»Өеҝ…йЎ»еңЁдҪ  100% жҺ§еҲ¶зҡ„иө„дә§дёҠи·‘(иҮӘе·ұзҡ„ VM / VPS / жөӢиҜ•зҪ‘ж®ө / жјҸжҙһиөҸйҮ‘жҺҲжқғиҢғеӣҙ)гҖӮ

дёҖгҖҒеҗҲжі•жҺҲжқғ:иҝҷдёҖз« иҜ»дёӢеҺ»еүҚеҝ…йЎ»жғіжё…жҘҡ вҖӢ

1.1 д»Җд№ҲеҸ«"жҺҲжқғ" вҖӢ

еҚ•зәҜеҸЈеӨҙеҗҢж„Ҹ вү  еҗҲжі•жҺҲжқғгҖӮеҗҲи§„жё—йҖҸжөӢиҜ•иҮіе°‘жңүд»ҘдёӢж–ҮжЎЈ:

1. SOW(Statement of Work,е·ҘдҪңиҜҙжҳҺд№Ұ)
   - жөӢиҜ•иҢғеӣҙ:е…·дҪ“зҡ„ IP ж®ө / еҹҹеҗҚ / еә”з”Ё / API
   - жөӢиҜ•ж—¶й—ҙзӘ—еҸЈ:еҮ зӮ№еҲ°еҮ зӮ№
   - дёҚеҸҜи§Ұзў°зҡ„иө„дә§:з”ҹдә§ж•°жҚ®еә“ / й«ҳ SLA жңҚеҠЎзӯү
   - жөӢиҜ•ж–№ејҸ:й»‘зӣ’ / зҒ°зӣ’ / зҷҪзӣ’
   
2. ROE(Rules of Engagement,дәӨжҲҳи§„еҲҷ)
   - е“Әдәӣж”»еҮ»жүӢжі•е…Ғи®ё(SQLi / XSS / з«ҜеҸЈжү«жҸҸ)
   - е“ӘдәӣзҰҒжӯў(зӨҫе·Ҙ / зү©зҗҶ / DoS жј”з»ғ)
   - еҮәзҺ°йЈҺйҷ©ж—¶еҰӮдҪ•дёӯжӯў
   - еә”жҖҘиҒ”зі»дәәз”өиҜқ

3. еҸҢж–№зӯҫеӯ—зӣ–з« зҡ„еҗҲеҗҢ
4. жөӢиҜ•ж–№дәәе‘ҳеҗҚеҚ• + еҮәеҸ‘ IP ж®ө

вҶ’ дёүиҖ…е…Ёжңү вҶ’ жүҚз®—жі•еҫӢж„Ҹд№үдёҠзҡ„жҺҲжқғ

жјҸжҙһиөҸйҮ‘е№іеҸ°(HackerOne / Bugcrowd)жҳҜеҸҰдёҖз§ҚеҗҲжі•жҺҲжқғвҖ”вҖ”е№іеҸ° + еҺӮе•ҶжҸҗеүҚе…¬е‘ҠдәҶ"ејҖж”ҫжөӢиҜ•иҢғеӣҙ",дҪ жҢүи§„еҲҷжҸҗдәӨеҚіеҸҜгҖӮ

1.2 зҒ°иүІ / зәўиүІеңәжҷҜеҜ№з…§ вҖӢ

з»ҝиүІ(еҗҲжі•):
  - иҮӘе·ұзҡ„ VM / VPS / е®¶йҮҢи·Ҝз”ұеҷЁ
  - е…¬еҸёз»ҷдҪ жҺҲжқғзҡ„жөӢиҜ•зҺҜеўғ(жңү SOW)
  - HackerOne / Bugcrowd йЎ№зӣ®еҶ…зҡ„иө„дә§
  - HTB / TryHackMe / VulnHub зӯүз»ғд№ йқ¶жңә
  - CTF жҜ”иөӣжҢҮе®ҡзҡ„йўҳзӣ®зҺҜеўғ

зҒ°иүІ(й«ҳйЈҺйҷ©,еҲ«зў°):
  - "зңӢиө·жқҘе…¬ејҖ"зҡ„жү«жҸҸ:е°ұз®— nmap дёҖдёӘе…¬зҪ‘ IP д№ҹиҝқжі•
  - "зңӢиө·жқҘж— е®і"зҡ„жҺўжөӢ:dig жІЎдәӢ,дҪҶзӣ®еҪ•зҲҶз ҙе·ІжҳҜж”»еҮ»иЎҢдёә
  - иҮӘе·ұжіЁеҶҢзҡ„жҹҗдёӘ SaaS жөӢиҜ•иҙҰеҸ·:еӨҡж•° SaaS жңҚеҠЎжқЎж¬ҫжҳҺзЎ®зҰҒжӯў
  - жңӢеҸӢиҜҙ"жҲ‘еҗҢж„ҸдҪ жү“"вҖ”вҖ”еҸЈеӨҙдёҚз®—,иҰҒзңӢе…¬еҸёжҳҜеҗҰжҺҲжқғд»–

зәўиүІ(зӣҙжҺҘиҝқжі•):
  - д»»дҪ•жңӘжҺҲжқғзҡ„з«ҜеҸЈжү«жҸҸ(дёӯеӣҪгҖҠеҲ‘жі•гҖӢ285)
  - д»»дҪ•жңӘжҺҲжқғзҡ„ејұеҜҶз Ғе°қиҜ•(еҗҢдёҠ)
  - жҠ“еҲ«дәәзҡ„зҪ‘з»ңжөҒйҮҸ(гҖҠзҪ‘з»ңе®үе…Ёжі•гҖӢ27 жқЎ)
  - жҠҠжөӢиҜ•з»“жһңеҸ‘еҲ°зҪ‘дёҠзӮ«иҖҖ(жү©ж•ЈдәӢж•…,еҠ йҮҚеӨ„зҪҡ)

з»ҸйӘҢжі•еҲҷ:дёҚиғҪ 100% иҜҒжҳҺ"жҲ‘жңүд№ҰйқўжҺҲжқғ"зҡ„иө„дә§,дёҖеҫӢдёҚеҠЁжүӢгҖӮиҝҷдёҖеҺҹеҲҷдҝқжҠӨдҪ зҡ„иҒҢдёҡз”ҹж¶ҜгҖӮ

1.3 йҳІеҫЎе·ҘзЁӢеёҲдёәд»Җд№ҲиҝҳиҰҒеӯҰ вҖӢ

дёҚжҮӮж”»еҮ»зҡ„йҳІеҫЎ = и’ҷзңјжҢЁжү“
  - дёҚзҹҘйҒ“ nmap й•ҝд»Җд№Ҳж ·,зңӢ IDS ж—Ҙеҝ—дёҖи„ёжҮө
  - дёҚзҹҘйҒ“ SQL жіЁе…Ҙзҡ„зңҹе®һ payload,WAF и§„еҲҷеҶҷеҫ—зЁҖ
  - дёҚзҹҘйҒ“ mitmproxy жҖҺд№ҲжҠ“ HTTPS,дёҡеҠЎж–№иҜҙ"жҲ‘д»¬з”ЁдәҶ SSL дёҚдјҡиў«жҠ“"вҖ”вҖ”дҪ еҲӨж–ӯдёҚеҮә
  - дёҚзҹҘйҒ“иҜҒд№Ұй’үжүҺиғҪиў«з»•,д»ҘдёәеҠ  SSL Pinning е°ұдёҮдәӢеӨ§еҗү

жҮӮж”»еҮ»зҡ„йҳІеҫЎ = жғіж•Ңд№ӢжүҖжғі
  - зңӢеҲ° SYN жү«жҸҸж—Ҙеҝ—з«ӢеҲ»еҲҶжөҒеҲ°иҜұйҘөз«ҜеҸЈ
  - WAF и§„еҲҷжҢүзңҹе®һ payload жЁЎејҸдјҳеҢ–
  - зҹҘйҒ“е“ӘдёӘеұӮи–„ејұ,еҠ еӣәж–№еҗ‘жңүжҚ®

еӯҰж”»еҮ»жҳҜжүӢж®ө,еҒҡйҳІеҫЎжҳҜзӣ®зҡ„вҖ”вҖ”жң¬з« иҮӘе§ӢиҮіз»ҲжҳҜиҝҷдёӘз«ӢеңәгҖӮ

  вҶ’ жңҚеҠЎз«Ҝжңүе®Ңж•ҙиҝһжҺҘж—Ҙеҝ—,еҸҚиҖҢжӣҙжҳҺжҳҫ

  вҶ’ йҳІеҫЎ:ејҖеҗҜ ICMP йҖҹзҺҮйҷҗеҲ¶

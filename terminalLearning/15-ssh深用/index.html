    UseKeychain yes(macOS) macOS дёҠ passphrase дёҚеӯҳ Keychain,жҜҸж¬ЎйҮҚеҗҜйғҪиҰҒйҮҚж–°иҫ“
    HashKnownHosts yes known_hosts жҳҺж–Ү,жі„йңІеҗҺзӯүдәҺе…¬ејҖдё»жңәеҲ—иЎЁ

    иҝҷдёҖж®өдёҖж—ҰеҶ…еҢ–,дҪ зңӢд»»дҪ• ssh config йғҪеғҸзңӢ Go еҮҪж•°зӯҫеҗҚдёҖж ·,дёҖзңјиғҪиҜ»жҮӮеңЁеЈ°жҳҺд»Җд№ҲгҖӮ

   еҗҺз»ӯ ssh devbox: еӨҚз”Ёдё»иҝһжҺҘ,зһ¬й—ҙ (~50ms)
   ControlPath socket ж–Үд»¶з»ҙжҢҒ 10 еҲҶй’ҹ

   йҷ„иө ж•Ҳжһң:scp / rsync / git push еҗҢдё»жңәж—¶д№ҹеӨҚз”Ё,дёҚйҮҚж–°жҸЎжүӢ

    жіЁж„Ҹ:ControlPath и·Ҝеҫ„йҮҢиҰҒжңү %r@%h:%p(user/host/port),дёҚз„¶дёҚеҗҢ user дјҡиў«й”ҷиҜҜеӨҚз”ЁгҖӮиҖҢдё” ~/.ssh/cm/ иҝҷдёӘзӣ®еҪ•иҰҒиҮӘе·ұ mkdir -p,ssh дёҚдјҡиҮӘеҠЁе»әгҖӮ

    дёүгҖҒProxyJump:жӣҝд»Ј ProxyCommand зҡ„зҺ°д»ЈеҶҷжі•

    3.1 иҖҒеҶҷжі• vs ж–°еҶҷжі•

    иҖҒеҶҷжі•(OpenSSH < 7.3,2016 е№ҙд№ӢеүҚ):
   ssh -o "ProxyCommand=ssh -W %h:%p bastion" target
   жҲ– config йҮҢ:
      Host target
          ProxyCommand ssh -W %h:%p bastion

ж–°еҶҷжі•(OpenSSH вүҘ 7.3,2025 й»ҳи®Ө):
   ssh -J bastion target
   жҲ– config йҮҢ:
      Host target
          ProxyJump bastion

    дёәд»Җд№ҲжҚў: - ProxyJump жҳҜ ssh еҚҸи®®зә§еҲ«ж”ҜжҢҒ,bastion дёҠдёҚйңҖиҰҒиЈ… nc(иҖҒ ProxyCommand з”Ё -W ж—¶е·Із»ҸдёҚйңҖиҰҒ,дҪҶжӣҙж—©зҡ„ nc %h %p еҶҷжі•йңҖиҰҒ) - ProxyJump иө°зҡ„жҳҜ direct-tcpip channel(еҚҸи®®еҶ…йҖҡйҒ“),жҜ” ProxyCommand иө·дёҖдёӘ ssh еӯҗиҝӣзЁӢжӣҙиҪ» - иҜӯжі•жӣҙзҹӯ,жҳ“иҜ»

    3.2 еӨҡи·і:-J host1,host2

  вҶ’ ж°ёиҝңеҸӘеңЁдҪ з¬”и®°жң¬дёҠжңүз§Ғй’Ҙ,иҝңз«ҜйҖҡиҝҮ ProxyJump д»ЈзҗҶзӯҫеҗҚ

  вҶ’ жҲ–з”Ё ProxyJump и®©иҝңз«Ҝзҡ„ git иө°жң¬ең° agent зӯҫеҗҚ

  вҶ’ иҝҷд»Ҫж–Үд»¶жҜҸиЎҢйғҪжҳҜдҪ зҡ„е·ҘзЁӢиө„дә§
  вҶ’ chezmoi / yadm зәіз®Ў,ж–°жңәеҷЁдёҖиЎҢеҗҢжӯҘ

вң— дёҚи®ҫ ServerAliveInterval
  вҶ’ зҪ‘з»ңдёҖжҠ–е°ұдёў session,vim еҚҠе°Ҹж—¶зҷҪе№І
  вҶ’ 60 / 3 жҳҜ 2026 зҡ„й»ҳи®Өй…ҚзҪ®

вң— дёҚз”Ё ControlMaster
  вҶ’ tmux 6 дёӘ pane еҗҢж—¶ ssh = 6 ж¬Ўе®Ңж•ҙжҸЎжүӢ = 10 з§’жҠ–еҠЁ
  вҶ’ ControlMaster auto + ControlPersist 10m жҳҜ 2026 й»ҳи®Ө

    еҚҒеӣӣгҖҒзңӢе®ҢиҝҷдёҖзҜҮдҪ еә”иҜҘиғҪ

    • еҶҷеҮәдёҖд»Ҫз”ҹдә§еҸҜз”Ёзҡ„ 70 иЎҢ ~/.ssh/configвҖ”вҖ”еёҰ ControlMasterгҖҒAddKeysToAgentгҖҒUseKeychain(macOS)гҖҒHashKnownHostsгҖҒaccept-newгҖҒProxyJumpгҖҒSetEnv,жҜҸдёҖиЎҢйғҪи®Іеҫ—жё…жҘҡдёәд»Җд№Ҳ
    • и§ЈйҮҠ ssh дёүз§Қз«ҜеҸЈиҪ¬еҸ‘(-L / -R / -D)зҡ„ж–№еҗ‘е’Ңе…ёеһӢеңәжҷҜвҖ”вҖ”иғҪеңЁзҷҪжқҝдёҠз”» ASCII еӣҫ,дёҚжҹҘж–ҮжЎЈ
    • и®ҫи®ЎдёҖдёӘ bastion жЁЎејҸвҖ”вҖ”client дҫ§дёҖиЎҢ ProxyJump bastion иҰҶзӣ–жүҖжңү prod-*;server дҫ§ sshd_config зҡ„зЎ¬жҖ§зәҰжқҹ(зҰҒ rootгҖҒзҰҒеҜҶз ҒгҖҒзҰҒ agent forwardгҖҒејәеҲ¶ keyгҖҒsession еҪ•еғҸ)
    • е®ҢжҲҗдёҖж¬ЎеҜҶй’Ҙз®ЎзҗҶж”№йҖ вҖ”вҖ”жҠҠ RSA 2048 жҚўжҲҗ ed25519,жҢүз”ЁйҖ”жӢҶ key,жҺҘдёҠ ssh-agent + Keychain жҲ– 1Password,еҒҡеҲ°"з§Ғй’ҘдёҚиЈёеҘ”гҖҒж–°жңәеҷЁиҝҒз§»жңүи·Ҝеҫ„"
    • еҲӨж–ӯд»Җд№Ҳж—¶еҖҷз”Ё sshгҖҒд»Җд№Ҳж—¶еҖҷз”Ё Tailscale / SSM / Cloudflare / TeleportвҖ”вҖ”з”Ёз¬¬еҚҒиҠӮзҡ„йҖүеһӢиЎЁиғҪз»ҷеӣўйҳҹе®ҡдёҖд»Ҫ"иҝңзЁӢи®ҝй—®ж ҮеҮҶ"
    • з”Ё rsync жӣҝжҚў scp,зҗҶи§Јдёәд»Җд№Ҳ rsync -avh --progress --partial жҳҜй»ҳи®Өе§ҝеҠҝ
    • йҒҝејҖжң¬ж–Үз¬¬еҚҒдёүиҠӮзҡ„ 14 жқЎеҸҚеҜ№еҶҷжі•вҖ”вҖ”иҝҷдәӣжҜҸжқЎйғҪжҳҜдәӢж•…жәҗ
    • з»ҷеӣўйҳҹж–°дәәеҶҷдёҖд»ҪгҖҢssh дёҠжүӢ checklistгҖҚ:5 еҲҶй’ҹз”ҹ keyгҖҒ10 еҲҶй’ҹж”№ configгҖҒ5 еҲҶй’ҹжөӢ ProxyJumpгҖҒ5 еҲҶй’ҹжөӢ LocalForward,30 еҲҶй’ҹеҶ…иғҪ ssh prod-db е№Іжҙ»

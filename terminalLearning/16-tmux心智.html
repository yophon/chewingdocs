1.3 tmux и§ЈеҶізҡ„дёӨд»¶дәӢ(дёҚжҳҜеҲҶеұҸ) вҖӢ

4.1 дёәд»Җд№Ҳеҝ…йЎ»жңү prefix вҖӢ

tmux и·‘еңЁз»Ҳз«ҜйҮҢ,з»Ҳз«ҜйҮҢжүҖжңүжҢүй”®йғҪиҰҒе…Ҳз»ҸиҝҮ shell / зЁӢеәҸвҖ”вҖ”еҰӮжһң tmux жғіиҰҒ"зӢ¬еҚ "еҝ«жҚ·й”®(жҜ”еҰӮзӣҙжҺҘжҢү Ctrl-T еҲҮ window),йӮЈ vim зҡ„ Ctrl-T(еӣһеҲ°дёҠдёҖдёӘ tag)е°ұеәҹдәҶ,Emacs зҡ„ C-x е…ЁеәҹдәҶ,bash зҡ„ C-r д№ҹеәҹдәҶгҖӮз»Ҳз«ҜжҢүй”®з©әй—ҙжҳҜе…ұдә«зҡ„,tmux дёҚиғҪзӢ¬еҚ гҖӮ

prefix зҡ„жң¬иҙЁжҳҜ"жЁЎејҸеҲҮжҚў":

жІЎжңү prefix ж—¶:
   дҪ зҡ„жҢүй”® вҶ’ з»Ҳз«Ҝ вҶ’ еҪ“еүҚеүҚеҸ°иҝӣзЁӢ(vim / shell / Claude / ...)
   
жҢүдәҶ prefix д№ӢеҗҺ(зҹӯжҡӮзҡ„"tmux жЁЎејҸ"):
   дҪ зҡ„жҢүй”® вҶ’ з»Ҳз«Ҝ вҶ’ tmux жӢҰжҲӘиҝҷдёҖдёӘ key
   tmux еӨ„зҗҶе®ҢдёҖдёӘ key еҗҺ,иҮӘеҠЁйҖҖеҮә "tmux жЁЎејҸ"
   дёӢдёҖдёӘжҢүй”®еҸҲеӣһеҲ°жӯЈеёёжөҒзЁӢ

иҝҷи·ҹ vim зҡ„ modal(normal / insert)жҳҜеҗҢдёҖдёӘеҝғжҷәвҖ”вҖ”з”ЁдёҖдёӘеүҚзјҖеҲҮеҲ°"е·Ҙе…·жЁЎејҸ",еҒҡдёҖд»¶дәӢ,иҮӘеҠЁеҲҮеӣһеҺ»гҖӮ

4.2 й»ҳи®Ө prefix жҳҜ Ctrl-B,дёәд»Җд№ҲеӨ§е®¶йғҪж”№ вҖӢ

й»ҳи®Ө:Ctrl-B
   - йЈҹжҢҮиҰҒдјёеҲ°й”®зӣҳдёӯеӨ®еҒҸе·Ұ
   - и·ҹ readline зҡ„ Ctrl-B(е…үж Үе·Ұз§»дёҖеӯ—з¬Ұ)еҶІзӘҒ
   - и·ҹ emacs зҡ„ Ctrl-B еҶІзӘҒ
   - еӨ§жӢҮжҢҮжҢү Ctrl + йЈҹжҢҮжҢү B,жүӢеҠҝеҲ«жүӯ

еёёи§Ғж”№жі•:
   в‘  Ctrl-A     screen зҡ„й»ҳи®Ө,и·ҹ readline зҡ„"иЎҢйҰ–"еҶІзӘҒ
                дҪҶ readline Ctrl-A з”Ёеҫ—е°‘(Home й”®еӨҹдәҶ)
                жҢүиө·жқҘжңҖиҲ’жңҚ:еӨ§жӢҮжҢҮ Ctrl + йЈҹжҢҮ A иҮӘз„¶иҗҪ
   
   в‘Ў Ctrl-Space и·ҹ IDE зҡ„"иЎҘе…Ё"еҶІзӘҒ(VS Code / IntelliJ й»ҳи®Ө)
                еҰӮжһңдҪ  IDE е…ҡдёҚиҰҒйҖү
   
   в‘ў ` (еҸҚеј•еҸ·)еҚ•й”®ж—  modifier,жҢүиө·жқҘжңҖеҝ«
                дҪҶеҸҚеј•еҸ·еңЁ shell йҮҢжҳҜе‘Ҫд»ӨжӣҝжҚў,з»ҸеёёиҜҜиҫ“е…Ҙ
                йҖӮеҗҲ"еҸҚеј•еҸ·е·Із»ҸдёҚз”Ё"зҡ„дәә
   
   в‘Ј дёҚж”№        еқҡжҢҒ Ctrl-B д№ҹеҸҜд»Ҙ,зҶҹдәҶе°ұзҶҹдәҶ
                дҪҶ 99% зҡ„жҲҗзҶҹз”ЁжҲ·йғҪж”№дәҶ

ејәзғҲе»әи®®:Ctrl-A(йҷӨйқһдҪ йҮҚеәҰ emacs з”ЁжҲ·)гҖӮCtrl-A жҳҜе·ҘзЁӢз•Ңзҡ„дәӢе®һж ҮеҮҶ,еҮ д№ҺжүҖжңү tmux ж•ҷзЁӢгҖҒжүҖжңү dotfiles д»“еә“й»ҳи®ӨйғҪжҳҜ Ctrl-AгҖӮеӯҰдәҶ Ctrl-A д№ӢеҗҺи·Ёз”өи„‘еҲҮжҚў,дҪ еңЁеҲ«дәәжңәеҷЁдёҠд№ҹиғҪз”Ё(еҫҲеӨҡдәәд№ҹжҳҜ Ctrl-A)гҖӮ

4.3 еҸҢеҮ» prefix жҠҠ prefix дј з»ҷзЁӢеәҸ вҖӢ

ж”№е®Ң prefix д№ӢеҗҺ,иҝҳиҰҒи§ЈеҶі**"жҲ‘зңҹзҡ„жғіз»ҷзЁӢеәҸеҸ‘ Ctrl-A жҖҺд№ҲеҠһ"**(жҜ”еҰӮ bash зҡ„"е…үж Үз§»еҲ°иЎҢйҰ–"жҲ– vim зҡ„жҹҗдәӣжҸ’д»¶)гҖӮ

# еңЁ .tmux.conf йҮҢ:

иҝҷж ·:Ctrl-A еҚ•зӢ¬жҢү = иҝӣе…Ҙ tmux жЁЎејҸ,Ctrl-A еҸҢеҮ» = жҠҠ Ctrl-A еҸ‘з»ҷеҪ“еүҚзЁӢеәҸгҖӮ

4.4 prefix жҳҜ"е®үе…Ёй—Ё",дёҚжҳҜ"йә»зғҰ" вҖӢ

ж–°дәәе«Ң prefix йә»зғҰжҳҜеӣ дёәжІЎдҪ“дјҡиҝҮе®ғйҳІдҪҸзҡ„дәӢж•…:

жІЎжңү prefix(еҒҮи®ҫ tmux зӣҙжҺҘз»‘ Ctrl-W жҳҜе…і pane):
   - дҪ еңЁ vim йҮҢжҢү Ctrl-W(еҲҮ window еӯҗжЁЎејҸ)
   - vim зңӢдёҚеҲ° Ctrl-W,еӣ дёә tmux жӢҰжҲӘдәҶ
   - дҪ зҡ„ pane жІЎдәҶ
   - vim зҡ„еҝ«жҚ·й”®е…Ёеәҹ
   - иЈ…дёҖдёӘ emacs?Ctrl-X д№ҹиў« tmux жҠў
   - иЈ… Claude Code?Ctrl-N(дёӢдёҖжқЎж¶ҲжҒҜ)иў« tmux жҠў

жңү prefix:
   - дҪ  vim йҮҢжҢү Ctrl-W вҶ’ жӯЈеёёеҲҮ vim еӯҗжЁЎејҸ
   - дҪ жғіе…і tmux pane вҶ’ prefix + x
   - жүҖжңүз»Ҳз«ҜзЁӢеәҸзҡ„еҝ«жҚ·й”® 100% дҝқз•ҷ
   - tmux и·ҹзЁӢеәҸ"дёҚжҠўй”®"

prefix зҡ„жң¬иҙЁжҳҜ"з§ҹз”ЁдёҖдёӘ key namespace"вҖ”вҖ”дҪ еҖҹдёҖдёӘ key(C-a)з»ҷ tmux,дҪңдёәеӣһжҠҘ,tmux дёҚжҠўд»»дҪ•е…¶д»– keyгҖӮиҝҷз¬”дәӨжҳ“йқһеёёеҲ’з®—гҖӮ

дә”гҖҒеҹәзЎҖж“ҚдҪңзҡ„"иҜӯд№үеҲҶз»„" вҖӢ

дёҚжҠ„еҝ«жҚ·й”®еӨ§е…Ё,и®ІеҝғжҷәгҖӮtmux зҡ„еҝ«жҚ·й”®зңӢдјјд№ұ(prefix еҗҺйқўи·ҹдёҖе Ҷеӯ—жҜҚ),дҪҶеҲҶз»„еҗҺеҸӘжңүдёүзұ»вҖ”вҖ”session ж“ҚдҪң / window ж“ҚдҪң / pane ж“ҚдҪң,еҠ дёҠ copy-mode дёҖзұ»е·Ҙе…·ж“ҚдҪңгҖӮ

5.1 Session зұ» вҖӢ

Session ж“ҚдҪң(ж•ҙдёӘе·ҘдҪңжөҒзә§еҲ«зҡ„дәӢ):

  tmux new -s work                  # е‘Ҫд»ӨиЎҢ:ж–°е»әдёҖдёӘеҸ« work зҡ„ session

  tmux kill-session -t work         # е‘Ҫд»ӨиЎҢ:жқҖжҺү work

  prefix + s                        # session еҶ…:дәӨдә’ејҸйҖү session еҲҮжҚў
  prefix + $                        # session еҶ…:йҮҚе‘ҪеҗҚеҪ“еүҚ session
  prefix + (    prefix + )          # session еҶ…:дёҠ/дёӢдёҖдёӘ session

8.3 "еұҸе№•е…ұдә«":еҗҢдёҖ session еӨҡдәә attach вҖӢ

tmux зҡ„еҸҰдёҖдёӘиў«дҪҺдј°зҡ„иғҪеҠӣ:еӨҡ client еҗҢж—¶ attach еҗҢдёҖдёӘ sessionвҖ”вҖ”жүҖжңүдәәзңӢеҲ°зҡ„еұҸе№•е®Ңе…ЁдёҖиҮҙ,жүҖжңүдәәзҡ„й”®зӣҳйғҪиғҪиҫ“е…Ҙ(еҚҸдҪңжЁЎејҸ)гҖӮ

bash
# дҪ  ssh иҝӣ server,иө·дёҖдёӘ session:
$ tmux new -s pair

# еҗҢдәӢд№ҹ ssh иҝӣеҗҢдёҖеҸ° server,attach:
$ tmux attach -t pair

# зҺ°еңЁдҪ дҝ©зңӢзҡ„жҳҜеҗҢдёҖдёӘеұҸе№•,дҪ жү“зҡ„еӯ—д»–иғҪзңӢи§Ғ,д»–жү“зҡ„еӯ—дҪ иғҪзңӢи§Ғ
# иҝҷжҳҜеӨ©з„¶зҡ„"еұҸе№•е…ұдә« + еҚҸдҪңзј–иҫ‘",дёҚйңҖиҰҒд»»дҪ•и§Ҷйў‘дјҡи®®иҪҜд»¶

зңҹе®һз”Ёжі•:

- иҝңзЁӢ pair programming (дҪ еҶҷд»Јз Ғд»–зңӢдҪ еҶҷ)
- иҝңзЁӢ debugging      (дҪ еҮәй—®йўҳи®© senior attach жқҘзңӢ)
- иҝңзЁӢж•ҷеӯҰ           (дҪ и·‘е‘Ҫд»Өз»ҷеӯҰз”ҹзңӢ,еӯҰз”ҹд№ҹиғҪи·‘з»ҷдҪ зңӢ)
- зҒҫйҡҫеә”жҖҘ           (дәӢж•…ж—¶ SRE еӨҡдәәе…ұеҗҢзҷ»дёҖеҸ°жңәеҷЁеҚҸдҪң)

жіЁж„Ҹе®үе…Ё:еӨҡдәә attach = еӨҡдәәжңүе®Ңж•ҙзҡ„ shell жҺ§еҲ¶жқғвҖ”вҖ”дҪ иғҪи·‘зҡ„е‘Ҫд»Өд»–йғҪиғҪи·‘гҖӮдёҚиҰҒеңЁз”ҹдә§ж•Ҹж„ҹжңәеҷЁдёҠйҡҸдҫҝи®©дәә attachгҖӮеҰӮжһңеҸӘжғі"зңӢдёҚиғҪж“ҚдҪң",з”Ё tmux зҡ„ readonly жЁЎејҸ(tmux attach -t pair -r)гҖӮ

иҝҷз§Қе·ҘдҪңжөҒзҡ„ж ёеҝғ:й•ҝжңҹжҢӮиҪҪзҡ„ session = дҪ зҡ„"е·ҘдҪңеҝ«з…§"вҖ”вҖ”дҪ жҚўз”өи„‘гҖҒжҚўең°зӮ№гҖҒжҚў IDE,еҸӘиҰҒиҝңз«ҜжңәеҷЁжІЎйҮҚеҗҜ,дҪ зҡ„ session еңЁйӮЈе„ҝгҖӮиҝҷжүҚжҳҜ tmux зҡ„зҒөйӯӮгҖӮ

д№қгҖҒеөҢеҘ— tmux:еёёи§Ғз—ӣзӮ№ вҖӢ

90% з”Ё tmux дёҖе№ҙд»ҘдёҠзҡ„дәәйғҪйҒҮеҲ°иҝҮеөҢеҘ—й—®йўҳ:

дҪ еңЁжң¬ең°з”өи„‘и·‘ tmux(prefix C-a)

9.2 дёүз§Қи§Јжі• вҖӢ

жңҖе№ІеҮҖгҖӮеҰӮжһңдҪ  95% зҡ„ tmux е·ҘдҪңйғҪеңЁиҝңз«Ҝ,жң¬ең°е°ұеҲ«еҘ—дёҖеұӮгҖӮжң¬ең°зӣҙжҺҘ iTerm еӨҡ tab + иҝңз«Ҝ tmux,е®Ңе…ЁеӨҹз”ЁгҖӮ

и§Јжі• 2:иҝңз«Ҝз”ЁдёҚеҗҢ prefix

bash
# иҝңз«ҜжңәеҷЁзҡ„ ~/.tmux.conf:
set -g prefix C-b               # иҝңз«Ҝз”Ё Ctrl-B(й»ҳи®Ө)
unbind C-a                       # и§ЈйҷӨ Ctrl-A
bind C-b send-prefix             # еҸҢеҮ» C-b жҠҠ C-b еҸ‘еҮәеҺ»

# жң¬ең°:з”Ё C-a(жҲ‘д»¬д№ӢеүҚж”№зҡ„)
# иҝңз«Ҝ:з”Ё C-b(еӣһеҲ°й»ҳи®Ө)

# иҝҷж ·дҪ жҢү C-a вҶ’ жң¬ең° tmux жҺҘ,жҢү C-b вҶ’ йҖҸдј еҲ°иҝңз«Ҝ tmux жҺҘ

иҝҷжҳҜжңҖе®һз”Ёзҡ„и§Јжі•вҖ”вҖ”жң¬ең°иҝңз«Ҝеҗ„з”ЁдёҖеҘ—,дә’дёҚеҶІзӘҒгҖӮ

# жң¬ең°зҡ„ .tmux.conf:
bind-key -n C-a send-prefix     # еҚ•еҮ» C-a йҖҸдј з»ҷиҝңз«Ҝ
bind-key C-a send-prefix         # prefix + C-a д№ҹйҖҸдј 

дҪҶиҝҷж ·жң¬ең° tmux е°ұз”ЁдёҚдәҶ prefix дәҶвҖ”вҖ”еҸӘиғҪиҝңз«Ҝз”ЁгҖӮиҝҷз§Қи§Јжі•еҫҲе°‘з”ЁгҖӮ

10.1 screen:еә”жҖҘз”Ё вҖӢ

дҪ•ж—¶з”Ё screen:
  - дҪ  ssh иҝӣдёҖеҸ°йҷҢз”ҹ Linux,жІЎиЈ… tmux,д№ҹжІЎжі• sudo
  - дҪ еҸӘжғі "detach дёҖдёӢ" и·‘й•ҝд»»еҠЎ,дёҚйңҖиҰҒеҲҶеұҸ
  - дҪ дҝ®еӨҚдёҖдёӘ 1995 е№ҙзҡ„жңҚеҠЎеҷЁ,еҸӘжңү screen

screen зҡ„дјҳзӮ№:
  вң“ Linux еҮ д№Һй»ҳи®ӨиЈ…(/usr/bin/screen)
  вң“ жһҒз®Җ,еҮ д№ҺжІЎй…ҚзҪ®жҲҗжң¬
  вң“ еҗҢж ·ж”ҜжҢҒ detach / attach
  
screen зҡ„зјәзӮ№:
  вң— еҮ д№ҺеҒңжӣҙ
  вң— й…ҚзҪ®иҜӯжі•еҘҮжҖӘ
  вң— еҲҶеұҸдҪ“йӘҢе·®
  вң— жІЎжңү copy-mode зҡ„ vi жЁЎејҸ
  вң— зҠ¶жҖҒж Ҹе®ҡеҲ¶иғҪеҠӣејұ
  вң— дёҚж”ҜжҢҒзҺ°д»Јй”®дҪҚ
  вң— еҮ д№ҺжІЎжҸ’д»¶з”ҹжҖҒ

е®һз”Ёе»әи®®:е№іж—¶еӯҰ tmux,зңӢи§Ғ screen д№ҹи®ӨиҜҶ
        з”ҹдә§жңәеҰӮжһңеҸӘжңү screen,зҹҘйҒ“ detach жҳҜ Ctrl-A + d
        й•ҝжңҹ screen з”ЁжҲ·еә”иҜҘиҝҒеҲ° tmux

10.2 tmux:дё»жөҒ вҖӢ

дҪ•ж—¶з”Ё tmux(еҮ д№ҺжүҖжңүзҺ°д»ЈеңәжҷҜ):
  - дҪ йңҖиҰҒ detach / attach
  - дҪ йңҖиҰҒеҲҶеұҸ
  - дҪ йңҖиҰҒдјҡиҜқжҢҒд№…еҢ–
  - дҪ йңҖиҰҒиҝңз«Ҝй•ҝд»»еҠЎ
  - дҪ йңҖиҰҒ pair programming / еұҸе№•е…ұдә«
  - дҪ зҡ„ .tmux.conf жғі sync еҲ°жүҖжңүжңәеҷЁ

tmux зҡ„дјҳзӮ№:
  вң“ жҙ»и·ғз»ҙжҠӨ(жңҖиҝ‘дёҖж¬ЎеӨ§зүҲжң¬ 2024 иҝҳжңү)
  вң“ й…ҚзҪ®ж–Үд»¶жё…жҷ°(.tmux.conf иҜӯжі•зӣҙжҺҘ)
  вң“ жҸ’д»¶з”ҹжҖҒжҲҗзҶҹ(TPM дёҖиЎҢе®үиЈ…)
  вң“ copy-mode ж”ҜжҢҒ vi й”®дҪҚ
  вң“ зҠ¶жҖҒж Ҹй«ҳеәҰеҸҜе®ҡеҲ¶
  вң“ зҺ°д»Јй”®дҪҚеҸӢеҘҪ
  вң“ еӨ§йғЁеҲҶ dotfiles д»“еә“д»Ҙ tmux дёәй»ҳи®Ө
  вң“ ж–ҮжЎЈ / Stack Overflow зӯ”жЎҲжһҒеӨҡ

tmux зҡ„зјәзӮ№:
  вң— й»ҳи®Өй…ҚзҪ®йңҖиҰҒж”№(prefix C-b / еӨҚеҲ¶ж“ҚдҪң / еҲҶеұҸй”®)
  вң— еӯҰд№ жӣІзәҝеңЁ copy-mode / mouse on зҡ„еҚҸи°ғ
  вң— еөҢеҘ—ж—¶зҡ„ prefix еҶІзӘҒ
  вң— зҠ¶жҖҒжҒўеӨҚйңҖиҰҒжҸ’д»¶(tmux-resurrect),дё”дёҚиғҪжҒўеӨҚиҝӣзЁӢеҶ…йғЁзҠ¶жҖҒ

е®һз”Ёе»әи®®:2026 й»ҳи®Өе°ұжҳҜе®ғ,жң¬зі»еҲ—д№ҹд»Ҙе®ғдёәдҫӢ

10.3 Zellij:еЈ°жҳҺејҸж–°з§Җ вҖӢ

дҪ•ж—¶з”Ё Zellij:
  - дҪ ж„ҝж„Ҹ"иҠұдёҖдёӘеӯҰд№ жӣІзәҝжҚўдёҖдёӘжӣҙеҘҪзҡ„й»ҳи®Ө"
  - дҪ еҸ—дёҚдәҶ tmux зҡ„ 50 дёӘ prefix еҝ«жҚ·й”®
  - дҪ е–ңж¬ўеЈ°жҳҺејҸй…ҚзҪ®(KDL)
  - дҪ жғіиҰҒ"ејҖз®ұеҚіз”Ё"зҡ„зҠ¶жҖҒж Ҹ / жө®еҠЁзӘ—еҸЈ
  
Zellij зҡ„дјҳзӮ№:
  вң“ ејҖз®ұеҚіз”Ё(й»ҳи®Өй…ҚзҪ®еҸҜз”Ё,еҮ д№ҺдёҚз”Ёж”№)
  вң“ зҠ¶жҖҒж Ҹ + жө®еҠЁзӘ—еҸЈеҶ…зҪ®
  вң“ еЈ°жҳҺејҸ layout(yaml/KDL жҸҸиҝ°зӘ—ж јз»“жһ„)
  вң“ Rust еҶҷзҡ„,жҖ§иғҪеҘҪ
  вң“ keybinding жҸҗзӨәеёёй©»еә•йғЁ(еҜ№ж–°дәәеҸӢеҘҪ)
  
Zellij зҡ„зјәзӮ№:
  вң— з”ҹжҖҒиҝңдёҚеҰӮ tmux(2020 иө·жӯҘ,з”ҹжҖҒи–„)
  вң— дёҚе°‘й«ҳзә§еңәжҷҜзјәеӨұ(SSH е…ұдә« attach дёҚеҰӮ tmux жҲҗзҶҹ)
  вң— иҮӘе®ҡд№үдҪҷең°е°Ҹ(ејәзәҰе®ҡзҡ„д»Јд»·)
  вң— дёҚе°‘жңәеҷЁжІЎиЈ…(brew install zellij жүҚжңү)
  вң— йғЁеҲҶжҸ’д»¶еҠҹиғҪ tmux е·Іжңүзҡ„,Zellij иҝҳеңЁеҒҡ

е®һз”Ёе»әи®®:Zellij жҳҜжҪңеҠӣиӮЎ,дҪҶ 2026 иҝҳдёҚеҲ°еҸ–д»Ј tmux зҡ„зЁӢеәҰ
        18 зҜҮдјҡдё“и®І Zellij vs tmux жҖҺд№ҲйҖү
        еҰӮжһңдҪ еҲҡеӯҰ multiplexer,еҸҜд»ҘдёӨдёӘйғҪиҜ• 1 е‘ЁеҶҚйҖү

10.4 йҖүеһӢз»“и®ә вҖӢ

й»ҳи®Ө:tmux

дҫӢеӨ–:
  - иҝңз«ҜжңәеҷЁеҸӘжңү screen,дҪ еҸӘжҳҜдёҙж—¶ detach вҶ’ screen
  - дҪ иҜ•иҝҮ tmux жғіжҚў Zellij,жңүж„ҸиҜҶең°иҜ„дј° вҶ’ Zellij(зңӢ 18 зҜҮ)
  - дҪ еқҡжҢҒеҸӘз”Ёз»Ҳз«ҜжЁЎжӢҹеҷЁзҡ„еҶ…зҪ®еҲҶеұҸ вҶ’ йӮЈе°ұдёҚеӯҰ tmux
    (дҪҶдҪ еӨұеҺ»дәҶ detach / attach,й•ҝд»»еҠЎиҮӘе·ұжғіеҠһжі•)

еҚҒдёҖгҖҒеҸҚеҜ№зҡ„еҶҷжі• вҖӢ

и®Іе®ҢжӯЈзЎ®зҡ„,еҲ—еҮәеҮ жқЎеёёи§Ғй”ҷиҜҜвҖ”вҖ”и§ҒеҲ°дәҶе°ұзҹҘйҒ“жҳҜй”ҷгҖӮ

11.1 з”Ё tmux еҪ“зӘ—еҸЈз®ЎзҗҶ вҖӢ

й”ҷзҡ„:
  жү“ејҖ tmux е°ұжҳҜдёәдәҶ"еҲҶ 4 дёӘзӘ—ж ј,е·ҰдёҠ vim,еҸідёҠ server,е·ҰдёӢ git,еҸідёӢ tail"
  з”Ё tmux жӣҝд»Ј i3 / yabai / Hyprland зҡ„зӘ—еҸЈз®ЎзҗҶ
  
й”ҷеңЁе“Ә:
  - tmux жҳҜз»Ҳз«Ҝ multiplexer,дёҚжҳҜ OS зӘ—еҸЈз®ЎзҗҶеҷЁ
  - tmux з®Ўзҡ„жҳҜ"з»Ҳз«ҜйҮҢзҡ„зӘ—ж ј",дёҚжҳҜ"еұҸе№•дёҠзҡ„зӘ—еҸЈ"
  - дҪ зҡ„жөҸи§ҲеҷЁгҖҒSlackгҖҒSpotify иҝҷдәӣйқһз»Ҳз«ҜзЁӢеәҸ,tmux е®Ңе…ЁдёҚз®Ў
  - дҪ жғі"зӘ—еҸЈз®ЎзҗҶ",з”Ё i3 / yabai(macOS)/ Hyprland(Wayland)
  
жӯЈзЎ®и®ӨзҹҘ:
  - tmux з®Ўз»Ҳз«ҜзӘ—ж ј
  - i3 / yabai з®Ў OS зӘ—еҸЈ
  - иҝҷдёӨдёӘдёңиҘҝеҸҜд»ҘеҸ еҠ ,дҪҶеҠҹиғҪдёҚйҮҚеҸ 

11.2 йј ж Ү + iTerm еӨҡ tab жӣҝд»Ј tmux вҖӢ

й”ҷзҡ„:
  "жҲ‘жңү iTerm,Cmd-T ејҖж–° tab,Cmd-D еҲҶеұҸ,Cmd-1/2/3 еҲҮ tab,еӨҹз”ЁдәҶ"
  
й”ҷеңЁе“Ә:
  - iTerm зҡ„ tab / split йғҪи·ҹ"е®ўжҲ·з«Ҝ"з»‘жӯ»
  - е…іжҺү iTerm зӘ—еҸЈ / йҮҚеҗҜ Mac вҶ’ е…ЁжІЎ
  - дҪ  ssh иҝӣиҝңз«Ҝ,iTerm зҡ„ tab еңЁиҝңз«ҜжІЎз”Ё(йӮЈжҳҜжң¬ең°зҡ„)
  - й•ҝд»»еҠЎи·‘еңЁ iTerm tab йҮҢ,iTerm crash вҶ’ й•ҝд»»еҠЎжӯ»
  - дҪ жғід»Һе®¶з”өи„‘жҺҘз»ӯе…¬еҸёз”өи„‘зҡ„ 4 дёӘ tab вҶ’ еҒҡдёҚеҲ°
  
жӯЈзЎ®и®ӨзҹҘ:
  - iTerm зҡ„ tab / split = и§Ҷи§үе®№еҷЁ(жң¬ең°)
  - tmux зҡ„ window / pane = е·ҘдҪңжөҒе®№еҷЁ(еҸҜиҝңз«Ҝ,еҸҜжҢҒд№…)
  - дёӨдёӘеұӮж¬Ў,iTerm жҳҜ"жҲ‘зңӢе“ӘеҸ°жңә",tmux жҳҜ"йӮЈеҸ°жңәдёҠжҢӮдәҶд»Җд№Ҳ"

11.3 дёҚеӯҰ copy-mode,еҸӘз”Ёйј ж Үж»ҡеҠЁ вҖӢ

й”ҷзҡ„:
  иЈ… tmux д№ӢеҗҺејҖдәҶ mouse on
  д»Ҙдёәиҝҷж ·ж»ҡиҪ®е°ұиғҪж»ҡ buffer
  ж»ҡиҪ®дёҖж»ҡеҸ‘зҺ°иҮӘеҠЁиҝӣ copy-mode дәҶ,дҪҶдёҚдјҡз”Ё / / n
  жҗңзҙўеҺҶеҸІиҝҳжҳҜз”Ё grep + йҮҚж–°и·‘е‘Ҫд»Ө
  
й”ҷеңЁе“Ә:
  - mouse on еҸӘжҳҜи®©дҪ "з”Ёйј ж Үиҝӣ copy-mode",дёҚжҳҜ"з”Ёйј ж Үе°ұиЎҢдәҶ"
  - зңҹжӯЈзҡ„еӨҚеҲ¶ / жҗңзҙўиҝҳжҳҜиҰҒз”Ё hjkl + / иҝҷеҘ—й”®дҪҚ
  - дёҚеӯҰ copy-mode зӯүдәҺеӨұеҺ» tmux ж»ҡеұҸ / жҗңзҙў / еӨҚеҲ¶иғҪеҠӣзҡ„ 80%
  
жӯЈзЎ®и®ӨзҹҘ:
  - copy-mode жҳҜ tmux еӯҰд№ жӣІзәҝзҡ„ж ёеҝғ,жүӣиҝҮеҺ»е°ұеҖј
  - дёҖеӨ©з”ЁзҶҹ,дёҖе‘ЁеҶ…еҢ–
  - дёҚеӯҰзҡ„д»Јд»·:жҜҸж¬ЎжғізңӢдёҠдёҖеұҸйғҪеҫ— detach + scrollback,еҸҚдәәзұ»

11.4 дёҖдёӘ session еЎһжүҖжңүдәӢ вҖӢ

й”ҷзҡ„:
  жү“ејҖ tmux д№ӢеҗҺеҶҚд№ҹдёҚејҖж–° session
  жүҖжңү window е…ЁеңЁ "0" иҝҷдёӘй»ҳи®Ө session йҮҢ
  
й”ҷеңЁе“Ә:
  - session жҳҜ"д»¶дәӢзҡ„е®№еҷЁ",дёҖдёӘ session еЎһ 5 д»¶дәӢ = 5 д»¶дәӢдә’зӣёе№Іжү°
  - дёҖд»¶дәӢзҡ„ vim еҙ©дәҶ,жүҖжңү session йҮҢе…¶е®ғзҡ„ window и·ҹзқҖйҮҚжҺ’
  - жғіе…ідёҖд»¶дәӢ,еҸӘиғҪ kill window дёҖдёӘдёӘжқҖ
  - prefix + s еҲҮ session дҪ“йӘҢе®Ңе…Ёз”ЁдёҚдёҠ
  
жӯЈзЎ®и®ӨзҹҘ:
  - дёҖд»¶дәӢдёҖдёӘ session,session жҳҜ"е·ҘдҪңжөҒйў—зІ’еәҰ"
  - prefix + s / tmux ls / tmux attach -t X д№Ӣй—ҙзҡ„еҲҮжҚўжүҚжҳҜ tmux зҡ„еёёжҖҒ

11.5 жҠ„дёҖй•ҝдёІ .tmux.conf дёҚзҹҘйҒ“дёҖеҚҠе№Іеҳӣ вҖӢ

й”ҷзҡ„:
  GitHub жҗң "tmux config",жүҫ star еӨҡзҡ„,зӣҙжҺҘ cp еҲ°иҮӘе·ұзҡ„ ~/.tmux.conf
  200 иЎҢй…ҚзҪ®,иҮӘе·ұзңӢдёҚжҮӮдёҖеҚҠ
  
й”ҷеңЁе“Ә:
  - еҲ«дәәзҡ„й…ҚзҪ®еҢ№й…ҚеҲ«дәәзҡ„е·ҘдҪңжөҒ,дёҚеҢ№й…ҚдҪ 
  - дҪ дёҚзҹҘйҒ“е“ӘдёҖиЎҢе№Іе•Ҙ,еҮәй—®йўҳж”№дёҚеҠЁ
  - 200 иЎҢйҮҢ 80% дҪ иҝҷиҫҲеӯҗдёҚдјҡз”Ё(tmux-yank-buffer / tmux-fingers / ...)
  - еҗҜеҠЁж…ўгҖҒз»ҙжҠӨз—ӣгҖҒдёҠжүӢзҙҜ
  
жӯЈзЎ®и®ӨзҹҘ:
  - 17 зҜҮдјҡз»ҷдёҖд»Ҫ 40-60 иЎҢзҡ„з”ҹдә§зә§й…ҚзҪ®,жҜҸиЎҢиҜҙжё…жҘҡдҪңз”Ё
  - жҠ„иҰҒжҠ„еҲ°"жҲ‘иғҪи§ЈйҮҠжҜҸдёҖиЎҢ",жҠ„дёҚеҠЁзҡ„еҲ

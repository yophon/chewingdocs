зәҝдёҠжңҚеҠЎ 502 жҠҘиӯҰгҖӮдҪ зңӢеҲ°зҡ„ 90% е·ҘзЁӢеёҲз¬¬дёҖеҸҚеә”жҳҜиҝҷж ·:жү“ејҖ IDE,grep й”ҷиҜҜз Ғ,зҝ»еә”з”Ёж—Ҙеҝ—,й—®"жҳҜдёҚжҳҜд»Јз Ғе“ӘйҮҢ panic дәҶ",жӢүзҫӨйҮҢеҗҢдәӢ review жңҖиҝ‘зҡ„ PRгҖӮ40 еҲҶй’ҹиҝҮеҺ»,еӨ§е®¶еңЁзҢңд»Јз Ғй—®йўҳ,з»“жһңе…¶е®һжҳҜдёҠжёё DNS жҠҠдҪ и§ЈеҲ°й”ҷзҡ„ IP дёҠвҖ”вҖ”иҖҢиҝҷд»¶дәӢ dig example.com дёҖжқЎе‘Ҫд»ӨгҖҒ3 з§’е°ұиғҪйӘҢеҮәжқҘгҖӮ

curl / dig / nc / mtr иҝҷеӣӣдёӘе·Ҙе…·зҡ„е…ізі»,е°ұеғҸ fzf / jq / grep / awk д№ӢдәҺж–Үжң¬з®ЎйҒ“вҖ”вҖ”е®ғд»¬дёҚжҳҜ"й«ҳзә§жҠҖиғҪ",жҳҜе·ҘзЁӢеёҲе…ҘиЎҢзҡ„еҸҠж јзәҝгҖӮcurl жҳҜ HTTP зҡ„ print,dig жҳҜ DNS зҡ„ print,nc жҳҜ TCP зҡ„ print,mtr жҳҜи·Ҝз”ұзҡ„ printгҖӮдјҡиҝҷеӣӣд»¶еҘ—,дҪ е°ұиғҪд»Һеә”з”ЁеұӮеҫҖдёӢжү“еӣӣеұӮ;дёҚдјҡ,дҪ е°ұеҸӘиғҪеңЁиҮӘе·ұзҡ„д»Јз ҒйҮҢеҸҚеӨҚ grepгҖӮ

дёҖеҸҘиҜқе…Ҳи®°дҪҸ:curl / dig / nc / mtr еӣӣд»¶еҘ—жҳҜзҪ‘з»ңеұӮзҡ„ print debugвҖ”вҖ”дёҚжҳҜ SRE д№ҹеҫ—дјҡгҖӮеҮә 5xx зҡ„з¬¬дёҖжӯҘж°ёиҝңжҳҜ dig + curl -v,дёҚжҳҜ git log дёҚжҳҜжҹҘд»Јз ҒгҖӮиҝҷжҳҜе·ҘзЁӢеёҲйқўеҜ№з”ҹдә§ж•…йҡңзҡ„иӮҢиӮүи®°еҝҶ,з»ғеҮәжқҘеҶҚд№ҹеӣһдёҚеҺ»гҖӮ

иҝҷдёҖзҜҮжӢҶејҖи®І:curl зҡ„жқҖжүӢ flag(-v / -w / --resolve / -H / --data-binary / -L / -b -c / --cert)гҖҒhttpie е’Ң xh д»Җд№Ҳж—¶еҖҷжҜ” curl ејә(JSON ж—¶д»Јзҡ„зҺ°д»Јжӣҝд»Ј)гҖҒdig зҡ„ 4 дёӘжЁЎејҸ(+short / @server / +trace / и®°еҪ•зұ»еһӢ)гҖҒnc жөӢиҝһйҖҡе’Ңиө·е°Ҹ serverгҖҒmtr зңӢи·Ҝз”ұдёўеҢ…гҖҒtcpdump е…Ҙй—Ё(еҸӘеҲ°иғҪжҠ“еҢ…зңӢ pcap йӮЈдёҖжӯҘ,ж·ұе…Ҙз•ҷз»ҷ networkLearning)гҖҒWireshark / Charles / mitmproxy жЎҢйқўдёүд»¶еҘ—гҖҒдёҖж®өжңҚеҠЎ 5xx зҡ„зңҹе®һжҺ’жҹҘе…ЁжөҒзЁӢгҖҒе’ҢдёҖз»„еҸҚеҜ№зҡ„еҶҷжі•гҖӮ

# жөӢиҝһйҖҡ(еӨҡдёӘз«ҜеҸЈдёҖиө·)
portcheck() {
  local host=$1; shift

  echo "=== DNS ===";    dig +short "$host"
  echo "=== HTTP ==="; curl -v -o /dev/null "$1" 2>&1 | grep -E "(Trying|HTTP/|< [A-Z])"
  echo "=== Timing ==="; ctime "$1"
}

зҺ°д»Јжӣҝд»Је“ҒйҖҹйҖҡ:rg / fd / bat / eza / dust / sd / hyperfine / delta вҖӢ

2015 е№ҙд№ӢеҗҺ,Hacker News дёҠжҜҸйҡ”дёҖе‘Ёе°ұеҶ’еҮәдёҖзҜҮгҖҢжҲ‘з”Ё Rust йҮҚеҶҷдәҶ XгҖҚвҖ”вҖ”X еҸҜиғҪжҳҜ catгҖҒеҸҜиғҪжҳҜ sedгҖҒеҸҜиғҪжҳҜ topгҖӮиҝҷз§Қеё–еӯҗ 80% жҳҜеҷӘеЈ°:жңүзҡ„дҪңиҖ…еҸӘжҳҜз»ғжүӢ RustгҖҒжңүзҡ„жҚўдәҶдёӘеҗҚеӯ—еҠ зӮ№еҪ©иүІгҖҒжңүзҡ„и§ЈеҶізҡ„жҳҜдҪңиҖ…иҮӘе·ұжғіиұЎеҮәжқҘзҡ„й—®йўҳгҖӮдҪҶеү©дёӢ 20% жҳҜзңҹзҡ„еҘҪдёңиҘҝвҖ”вҖ”дёҖе№ҙеҶ…иҝӣе…Ҙдё»жөҒе·ҘзЁӢеёҲзҡ„ dotfilesгҖҒиў« GitHub еҮ дёҮжҳҹи®ӨеҸҜгҖҒзңҹзҡ„жӣҝд»ЈдәҶжҹҗдёӘдј з»ҹе·Ҙе…·зҡ„дҪҚзҪ®гҖӮиҝҷдёҖзҜҮе°ұжҳҜжҠҠиҝҷ 20% жҢ‘еҮәжқҘ,и®Іжё…жҘҡе®ғд»¬еҘҪеңЁе“ӘгҖҒд»Җд№Ҳж—¶еҖҷиҜҘжҚўгҖҒд»Җд№Ҳж—¶еҖҷдёҚиғҪжҚўгҖӮ

иҝҷдёҖжіўе·Ҙе…·зҡ„е…ұеҗҢзү№еҫҒ:Rust еҶҷгҖҒSIMD/е№¶иЎҢеҠ йҖҹгҖҒй»ҳи®Өе°ҠйҮҚ .gitignoreгҖҒеҪ©иүІеҜ№йҪҗгҖҒи·Ёе№іеҸ°дёҖд»ҪдәҢиҝӣеҲ¶вҖ”вҖ”жӢҝеҮәжқҘи·ҹдј з»ҹ Unix е·Ҙе…·ж”ҫеңЁдёҖиө·,дәәзңјдёҖзңӢе°ұзҹҘйҒ“е“ӘдёӘжҳҜ 2026 зҡ„гҖӮдҪҶгҖҢзҺ°д»ЈгҖҚдёҚзӯүдәҺгҖҢжӣҝжҚўгҖҚ:дҪ зҡ„и„ҡжң¬ж°ёиҝңеҸӘиғҪдҫқиө– POSIX,SSH иҝӣйҷҢз”ҹжңәеҷЁж°ёиҝңеҸӘжңү grep / find / cat,е®№еҷЁйҮҢ Alpine й•ңеғҸж°ёиҝңдёҚдјҡйў„иЈ… rgгҖӮиҝҷдёҖзҜҮзҡ„ж ёеҝғз«ӢеңәжҳҜ:еӯҰж–°зҡ„еҪ“ daily driver,з•ҷж—§зҡ„еҪ“ fallbackвҖ”вҖ”дёҚжҳҜгҖҢе…ЁйғЁ alias жҲҗж–°е·Ҙе…·гҖҚзҡ„зӣІзӣ®еҚҮзә§гҖӮ

дёҖеҸҘиҜқе…Ҳи®°дҪҸ:Rust CLI е·Ҙе…·жҷ®йҒҚжҜ”дј з»ҹ Unix е·Ҙе…·еҝ« 5-10Г—гҖҒUX еҘҪдёҖжЎЈгҖҒи·Ёе№іеҸ°,дҪҶдёҚжҳҜж— и„‘жӣҝжҚўвҖ”вҖ”дҪ зҡ„и„ҡжң¬д»Қз„¶еҸӘиғҪдҫқиө– POSIX,SSH иҝӣйҷҢз”ҹжңәеҷЁд»Қз„¶еҸӘжңү ls / grep / findгҖӮеӯҰж–°зҡ„еҪ“ daily driver,з•ҷж—§зҡ„еҪ“ fallbackгҖӮ

иҝҷдёҖзҜҮжӢҶејҖжқҘи®І:иҝҷдёҖжіў Rust CLI е·Ҙе…·жҷ®йҒҚеҝ«зҡ„ж №еӣ гҖҒж ёеҝғ 10 дёӘе·Ҙе…·йҖҹеҜ№иЎЁгҖҒrg / fd / bat / eza / dust / sd / hyperfine / delta зҡ„жңҖе°ҸеҸҜз”Ёй…ҚзҪ®гҖҒжҖҺд№ҲдёҚи®©"ж–°е·Ҙе…·дёҠзҳҫз—Ү"еҸ‘дҪңгҖӮдёҚжҠ„ READMEвҖ”вҖ”README еңЁйӮЈе„ҝ,иҝҷйҮҢеҸӘиЎҘ README жІЎи®ІйҖҸзҡ„гҖҢдёәд»Җд№ҲгҖҚе’ҢгҖҢд»Җд№Ҳж—¶еҖҷдёҚиҰҒиЈ…гҖҚгҖӮ

1.2 gitignore й»ҳи®Өе°ҠйҮҚвҖ”вҖ”ејҖеҸ‘иҖ…дёҚз”ЁеҶҚи§ЈйҮҠ вҖӢ

ж–°дәәз¬¬дёҖж¬Ўз”Ё grep еңЁеӨ§д»“еә“йҮҢжҗңд»Јз Ғ:

# зҝ»дёҚеҲ°зңҹжӯЈжғіиҰҒзҡ„йӮЈдёҖиЎҢ

з„¶еҗҺж–°дәәеҺ»жҗңгҖҢgrep жҖҺд№ҲеҝҪз•Ҙ node_modulesгҖҚ,еӯҰеҲ°дёҖе Ҷ --exclude-dir зҡ„еҶҷжі•вҖ”вҖ”дҪҶжҜҸж¬ЎйғҪиҰҒеҶҷдёҖйҒҚгҖӮrg 'foo' й»ҳи®Өе°ұи·іиҝҮжүҖжңү .gitignore йҮҢеҲ—еҮәзҡ„зӣ®еҪ•вҖ”вҖ”еҜ№ејҖеҸ‘иҖ…жқҘиҜҙ,иҝҷжҳҜ"еҮҸе°‘и®ӨзҹҘиҙҹжӢ…"зҡ„е·ҘзЁӢиғңеҲ©гҖӮ

rg / fd / dust / tokei йғҪй»ҳи®Өе°ҠйҮҚ:
   .gitignore        вҶҗ git й…Қзҡ„
   .ignore           вҶҗ иҝҷдәӣе·Ҙе…·зү№жңүзҡ„"йҖҡз”Ё ignore"
   .rgignore         вҶҗ rg зү№жңү
   .fdignore         вҶҗ fd зү№жңү
   е…ЁеұҖ ~/.config/git/ignore

иҝҷеҘ—"еҲҶеұӮ ignore"жҳҜ BurntSushi и®ҫи®Ўзҡ„дәӢе®һж ҮеҮҶвҖ”вҖ”еҗҺжқҘзҡ„е·Ҙе…·еҹәжң¬йғҪи·ҹгҖӮеҜ№жҜ”дј з»ҹ grep:жҜҸж¬ЎжүӢеҶҷ --exclude-dir={node_modules,dist,.git},йҮҚеӨҚеҠіеҠЁгҖӮ

1.4 иҒӘжҳҺй»ҳи®Ө = еҮҸе°‘ flag еҝғжҷә вҖӢ

grep 'foo' .              вҶҗ жҠҘй”ҷ,grep дёҚйҖ’еҪ’й»ҳи®Ө
grep -r 'foo' .           вҶҗ иҰҒеҠ  -r
rg 'foo'                  вҶҗ й»ҳи®ӨйҖ’еҪ’

find . -name '*.py'       вҶҗ й»ҳи®Өе…ЁеҗҚеҢ№й…Қ,иҰҒеҶҷ glob
fd '.py$'                 вҶҗ й»ҳи®ӨжӯЈеҲҷ,дҪҶеҢ№й…Қ path еӯҗдёІ

find . -name 'foo' -type f -exec grep 'bar' {} \;

rg 'bar' -g '*foo*'       вҶҗ rg зӣҙжҺҘжҗһе®ҡ

"еҮҸе°‘ flag еҝғжҷә"жҳҜиҝҷжіўе·Ҙе…·зҡ„ж ёеҝғ UX иғңеҲ©вҖ”вҖ”дҪ дёҚз”Ёи®° -r иҝҳжҳҜ -RгҖҒ--exclude иҝҳжҳҜ --exclude-dirгҖҒfind -name иҝҳжҳҜ find -path,е·Ҙе…·её®дҪ йҖүеҘҪдәҶ 95% еңәжҷҜдёӢзҡ„еҗҲзҗҶй»ҳи®ӨгҖӮ

# в”Җв”Җв”Җ жүҫеҲ°еҗҺе№Іеҳӣ(-x жҳҜ xargs,-X жҳҜжү№йҮҸ) в”Җв”Җв”Җ

{} / {.} / {/} йҖҹжҹҘ:

{}      е®Ңж•ҙи·Ҝеҫ„,e.g. ./src/foo.py
{.}     еҺ»жҺүжү©еұ•еҗҚ:./src/foo
{/}     basename:foo.py
{//}    dirname:./src
{/.}    basename еҺ»жү©еұ•еҗҚ:foo

иҝҷдәӣе’Ң GNU parallel зҡ„иҜӯжі•еҜ№йҪҗ,и®°дёҖеҘ—йҖҡз”ЁгҖӮ

7.4 sd жҜ” sed е®үе…ЁеңЁе“Ә вҖӢ

1. й»ҳи®Өе…Ёж–ҮеҢ№й…Қ,дёҚеҲҶиЎҢ
   sed й»ҳи®ӨжҢүиЎҢеӨ„зҗҶ,и·ЁиЎҢжӣҝжҚўиҰҒ -z жҲ–иҖ…еӨҚжқӮзҡ„ N е‘Ҫд»Ө
   sd й»ҳи®Өж•ҙдёӘж–Үд»¶еҪ“дёҖдёӘеӯ—з¬ҰдёІзңӢ

2. Unicode й»ҳи®ӨеҜ№
   sed еҜ№ Unicode зҡ„ \b / \w еңЁдёҚеҗҢ locale иЎЁзҺ°дёҚеҗҢ
   sd й»ҳи®Ө Unicode зұ»еҜ№йҪҗ Rust regex,иЎҢдёәдёҖиҮҙ

3. иҪ¬д№үе°‘,жҚ•иҺ·з»„з”Ё $1
   sed:  s/\(foo\)/\1-bar/    иҰҒиҪ¬д№ү ( )
   sd:   '(foo)' '$1-bar'     дёҚиҪ¬д№ү

4. й”ҷиҜҜжӣҙжё…жҘҡ
   sed й”ҷдәҶз»Ҹеёёйқҷй»ҳ,sd й”ҷдәҶзӣҙжҺҘжҠҘжё…жҘҡ

   sed -i еңЁ macOS иҰҒ '' еҚ дҪҚ,Linux дёҚиҰҒ,и·Ёе№іеҸ°еқ‘
   sd дёҖиҮҙиЎҢдёә

7.5 дёҚжӣҝд»Ј sed зҡ„жғ…еҶө вҖӢ

sd жҳҜжӣҝжҚўе·Ҙе…·,дёҚжҳҜжөҒзј–иҫ‘еҷЁвҖ”вҖ”sed иғҪеҒҡзҡ„еҫҲеӨҡдәӢ sd дёҚеҒҡ:

1. иЎҢеҸ·иҢғеӣҙзј–иҫ‘
   sed '5,10d' file.txt              # еҲ з¬¬ 5-10 иЎҢ вҖ”вҖ” sd жІЎиҝҷдёӘ
   sed '/^#/d' file.txt              # еҲ жүҖжңү # ејҖеӨҙзҡ„иЎҢ вҖ”вҖ” sd жІЎиҝҷдёӘ

2. еӨҚжқӮең°еқҖеҢ№й…Қ
   sed '/BEGIN/,/END/d'              # еҲ  BEGIN еҲ° END д№Ӣй—ҙ вҖ”вҖ” sd жІЎиҝҷдёӘ

3. еӨҡе‘Ҫд»Өз»„еҗҲ

4. иЎҢзј–иҫ‘ / жҸ’е…Ҙ / иҝҪеҠ 
   sed '5a\hello'                    # з¬¬ 5 иЎҢеҗҺжҸ’ hello вҖ”вҖ” sd жІЎиҝҷдёӘ

5. иҝңз«Ҝ / и„ҡжң¬е…је®№

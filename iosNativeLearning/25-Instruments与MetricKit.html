Instruments дёҺ MetricKit вҖӢ

24 зҜҮи®ІдәҶ SwiftUI еҶ…йғЁжҖ§иғҪиҝҪиёӘ,иҝҷдёҖзҜҮи®ІзәҝдёӢ + зәҝдёҠзҡ„дёӨеҘ—е·Ҙе…·:Instruments жҳҜејҖеҸ‘жңҹжң¬ең° profile;MetricKit жҳҜзәҝдёҠд»Һз”ЁжҲ·и®ҫеӨҮиҮӘеҠЁдёҠжҠҘзҡ„жҖ§иғҪжҢҮж Ү + еҙ©жәғж•°жҚ®гҖӮдёӨиҖ…еҗ„з®ЎдёҖж®ө,еҒҡзәҝдёҠ App йғҪеҝ…йЎ»з”ЁгҖӮ

дёҖеҸҘиҜқе…Ҳи®°дҪҸ:Instruments жҳҜ Xcode иҮӘеёҰзҡ„ profilerвҖ”вҖ”Time Profiler зңӢ CPU зҒ«з„°еӣҫгҖҒAllocations зңӢеҶ…еӯҳеҲҶй…ҚгҖҒLeaks зңӢжі„жјҸ(дҪҶжңүеұҖйҷҗ,дёҚжҠ“ retain cycle)гҖҒSwiftUI template зңӢ body и°ғз”ЁгҖӮMetricKit(MXMetricPayload)жҳҜ iOS зі»з»ҹжҜҸеӨ©жү“еҢ…дёҖд»Ҫз”ЁжҲ·и®ҫеӨҮзҡ„жҖ§иғҪж•°жҚ®иҮӘеҠЁз»ҷдҪ  App,зңӢдёҚеҲ°е…·дҪ“з”ЁжҲ·дҪҶжңүиҒҡеҗҲжҢҮж ҮгҖӮ

дёҖгҖҒInstruments жЁЎжқҝйҖүеһӢ вҖӢ

жү“ејҖ Instruments(Xcode вҶ’ Open Developer Tool вҶ’ Instruments,жҲ– вҢҳ + I)гҖӮеёёз”ЁжЁЎжқҝ:

жЁЎжқҝи§ЈеҶід»Җд№Ҳ

AllocationsеҶ…еӯҳеҲҶй…Қ,зңӢе“Әдәӣзұ»еҚ еҶ…еӯҳжңҖеӨҡ
LeaksеҶ…еӯҳжі„жјҸжЈҖжөӢ
SwiftUIView body йў‘зҺҮгҖҒRender passes
Animation HitchesеҚЎеё§еҲҶжһҗ(жҺүеё§еҺҹеӣ )

Core MLML жЁЎеһӢжҺЁзҗҶж—¶й—ҙгҖҒи°ғеәҰеҲ° CPU/GPU/ANE

з”Ё Profile build(Cmd + I)иҖҢдёҚжҳҜ DebugвҖ”вҖ”Debug дјҳеҢ–зә§еҲ« -Onone,жҖ§иғҪж•°жҚ®дёҚзңҹе®һ;Profile жҳҜ Release й…ҚзҪ®,д»ЈиЎЁзңҹе®һиҝҗиЎҢжғ…еҶөгҖӮ

дәҢгҖҒTime Profiler:CPU зҒ«з„°еӣҫ вҖӢ

  1. еҗҜеҠЁ App,еңЁ Instruments йҮҢйҖү Time Profiler
  2. Record(зәўиүІжҢүй’®),ж“ҚдҪң App и§ҰеҸ‘еҚЎйЎҝеңәжҷҜ
  3. Stop,зңӢе·ҰдёӢи§’и°ғз”Ёж Ҳ

зҒ«з„°еӣҫиҜ»жі•:

  • жЁӘе®Ҫ = еҚ з”Ё CPU ж—¶й—ҙжҜ”дҫӢвҖ”вҖ”и¶Ҡе®Ҫи¶ҠиҖ—ж—¶
  • зәөж·ұ = и°ғз”Ёж Ҳж·ұеәҰвҖ”вҖ”дёҠйқўжҳҜйЎ¶еұӮеҮҪж•°,дёӢйқўжҳҜеҸ¶еӯҗеҮҪж•°
  • жүҫжңҖе®Ҫзҡ„еҮҪж•°вҖ”вҖ”йҖҡеёёжҳҜжҖ§иғҪе…ғеҮ¶
  • иҝҮж»Өдё»зәҝзЁӢвҖ”вҖ”е·ҰдёӢи§’ Choose Target вҶ’ Main Thread

дҫӢ:дҪ зңӢеҲ° NoteRow.body еҮҪж•°дёӢйқўжңүдёҖзүҮе®Ҫзҹ©еҪўжҳҜ DateFormatter.string(from:),иҜҙжҳҺжҜҸж¬ЎйҮҚз®— body йғҪйҮҚж–° format date,дјҳеҢ–зӮ№е°ұеңЁиҝҷйҮҢ(static formatter)гҖӮ

дёүгҖҒAllocations:еҶ…еӯҳеҲҶеёғ вҖӢ

еҗҜеҠЁ Allocations,еҒҡе®ҢдёҖзі»еҲ—ж“ҚдҪңеҗҺ:

  • All Heap Allocations:иҝҳжҙ»зқҖзҡ„еҜ№иұЎжҖ»ж•°
  • All Heap & Anonymous VM:еҠ дёҠиҷҡжӢҹеҶ…еӯҳжҳ е°„
  • Persistent Bytes:д»Һжҹҗж—¶зӮ№ејҖе§ӢзҙҜз§Ҝзҡ„ж–°еҲҶй…Қ

Mark Generation еҠҹиғҪ:еңЁжҹҗдёӘзҠ¶жҖҒдёӢ"жү“дёӘж Үи®°",з„¶еҗҺж“ҚдҪң,зңӢж–°еўһдәҶе“ӘдәӣеҜ№иұЎвҖ”вҖ”е…ёеһӢз”ЁдәҺ"еҸ‘зҺ°е“ӘдәӣеҜ№иұЎжң¬иҜҘеӣһж”¶дҪҶжІЎеӣһж”¶"гҖӮ

дҫӢ:

  1. App жү“ејҖ,Mark Generation A
  2. иҝӣе…Ҙ Detail и§Ҷеӣҫ,еҒҡеҮ ж¬Ўж“ҚдҪң,Mark Generation B

еӣӣгҖҒLeaks зҡ„еұҖйҷҗ вҖӢ

Leaks е·Ҙе…·жЈҖжөӢе·Із»Ҹе®Ңе…ЁжІЎеј•з”ЁгҖҒдҪҶ ARC жІЎеӣһж”¶зҡ„еҜ№иұЎвҖ”вҖ”е…ёеһӢеңәжҷҜ:еҫӘзҺҜеј•з”ЁгҖӮ

swift
class A {
    var b: B?
}

class B {
    var a: A?      // вқҢ ејәеј•з”Ё,A.b вҶ’ B,B.a вҶ’ A,еҫӘзҺҜ
}

let a = A()
let b = B()
a.b = b

// зҰ»ејҖ scope еҗҺдёӨдёӘйғҪдёҚйҮҠж”ҫ

Leaks еҸӘиғҪжҠ“"е®Ңе…ЁжІЎеј•з”Ё"зҡ„еҫӘзҺҜвҖ”вҖ”жҹҗдәӣеңәжҷҜдёӢ Leaks зңӢдёҚеҲ°жі„жјҸ,еӣ дёәе®ғд»¬иҝҳиў«жҹҗдёӘ scope ејұејұең°иҝһзқҖ(жҜ”еҰӮ NotificationCenter)гҖӮ

  • еј№еҮәеҸҜи§ҶеҢ–еҜ№иұЎеӣҫ,еҸҜз–‘еҜ№иұЎ(иҝҮжңҹ view controller / store)ж—Ғиҫ№жңүзҙ«иүІ ! ж Үи®°

  • Disk Writes:еҶҷзӣҳеӨҡеҜјиҮҙз”өйҮҸж¶ҲиҖ—еӨ§

д№қгҖҒMetricKit:иҮӘеҠЁдёҠжҠҘ вҖӢ

MXMetricManager иҮӘеҠЁжӢҝзі»з»ҹзә§иҒҡеҗҲжҢҮж Ү + crash diagnostics:

swift
import MetricKit

final class MetricsHandler: NSObject, MXMetricManagerSubscriber {
    static let shared = MetricsHandler()
    
    func start() {
        MXMetricManager.shared.add(self)
    }
    
    func didReceive(_ payloads: [MXMetricPayload]) {
        for payload in payloads {
            // дёҖд»Ҫ payload жҳҜиҝҮеҺ» 24 е°Ҹж—¶зҡ„иҒҡеҗҲж•°жҚ®
            let appLaunchTime = payload.applicationLaunchMetrics?.histogrammedTimeToFirstDraw
            let cpu = payload.cpuMetrics?.cumulativeCPUTime
            let hangs = payload.applicationResponsivenessMetrics?.histogrammedApplicationHangTime
            // дёҠжҠҘеҲ°дҪ зҡ„жңҚеҠЎеҷЁ
            upload(payload.jsonRepresentation())
        }
    }
    
    func didReceive(_ payloads: [MXDiagnosticPayload]) {
        for payload in payloads {
            // crash / disk write / hang / cpu / app launch diagnostic
            for crash in payload.crashDiagnostics ?? [] {
                let callStack = crash.callStackTree.jsonRepresentation()
                upload(crash: callStack)
            }
        }
    }
}

MetricKit зҡ„зү№зӮ№:

  • зі»з»ҹжҜҸеӨ©дёҖж¬Ўжү“еҢ…ж•°жҚ®дј з»ҷ App(App еҗҜеҠЁж—¶)
  • ж•°жҚ®еҢҝеҗҚ(дёҚеёҰз”ЁжҲ·ж ҮиҜҶ)
  • еҢ…еҗ« crash / hang / cpu / disk / network е…ЁйқўиҜҠж–ӯ

MXDiagnosticPayload еҢ…еҗ«е…·дҪ“зҡ„ crash call stack,еҸҜд»ҘдёҠдј еҲ°иҮӘе·ұеҗҺз«ҜеҒҡиҒҡеҗҲеҲҶжһҗвҖ”вҖ”жҜ” Apple иҮӘеёҰ Organizer жӣҙзҒөжҙ»(иҮӘе®ҡд№үзңӢжқҝгҖҒжҢү build / и®ҫеӨҮеһӢеҸ·иҝҮж»Ө)гҖӮ

  • дёҚйҳ»еЎһдё»зәҝзЁӢ(ејӮжӯҘ)
  • еҸҜй…ҚзҪ® privacy(й»ҳи®Өж•Ҹж„ҹдҝЎжҒҜи„ұж•Ҹ)
  • еҸҜиў« Console.app е®һж—¶жҹҘзңӢ
  • иў« OS иҮӘеҠЁж”¶йӣҶеҲ° sysdiagnose

iOS 15+ жҸҗдҫӣ OSLogStore,еңЁ App еҶ…жЈҖзҙўжң¬и®ҫеӨҮзҡ„иҝҮеҺ»ж—Ҙеҝ—:

swift
import OSLog

func loadRecentLogs() async throws -> [String] {
    let store = try OSLogStore(scope: .currentProcessIdentifier)
    let position = store.position(timeIntervalSinceLatestBoot: 1)  // д»ҺеҗҜеҠЁеҗҺ 1s ејҖе§Ӣ
    
    let entries = try store.getEntries(at: position)
        .compactMap { $0 as? OSLogEntryLog }
        .filter { $0.subsystem == "com.example.NotesIsland" }
        .map { "\($0.date): \($0.composedMessage)" }
    
    return entries.suffix(100).map { String($0) }
}

# 函数与多返回值

> **导读**：函数在 Go 中是一等公民。Go 没有异常捕获机制，因此极其依赖多返回值来传递错误。

## 一、基础函数与多返回值
Go 原生支持多返回值，这也是处理 error 的基石。
```go
// 返回结果和错误
func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

func main() {
    res, err := divide(10, 2)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println(res)
}
```

## 二、命名返回值
可以在函数签名中直接命名返回值，在函数体中直接给它们赋值，最后使用空的 `return` 即可。
```go
func rectangleStats(w, h int) (area int, perimeter int) {
    area = w * h
    perimeter = 2 * (w + h)
    return // 自动返回 area 和 perimeter
}
```

## 三、闭包与匿名函数
```go
func counter() func() int {
    count := 0
    // 返回一个匿名函数，形成闭包，捕获了外部变量 count
    return func() int {
        count++
        return count
    }
}

func main() {
    c := counter()
    fmt.Println(c()) // 1
    fmt.Println(c()) // 2
}
```
> **避坑**：在老版本 Go 的 for 循环中使用 goroutine 执行闭包时，经常因为捕获了循环变量导致所有 goroutine 打印同一个值。Go 1.22 已经修复了循环变量的语义，但在老版本代码中依然需要注意。

# 基础语法速通

> **导读**：变量、数据类型、控制流与 defer。Go 抛弃了无意义的语法糖，回归编程本质。

## 一、变量与常量
Go 是静态强类型语言，但支持类型推导。
```go
// 1. 显式声明
var a int = 10
var s string = "hello"

// 2. 类型推导（推荐）
var b = 20

// 3. 简短声明（仅限函数内部使用，最常用）
c := 30
name := "gopher"

// 4. 常量与 iota 枚举
const Pi = 3.14
const (
    StatusOk = iota // 0
    StatusErr       // 1
)
```

## 二、控制流
Go 中只有 `for` 循环，没有 `while`。
```go
// 1. If-Else (支持前置初始化语句)
if x := compute(); x > 10 {
    fmt.Println("x is greater than 10")
}

// 2. For 循环
for i := 0; i < 5; i++ { }

// 模拟 while
n := 0
for n < 5 { n++ }

// 无限循环
for { break }

// 3. Switch (默认自带 break，不需要手动写)
switch os := runtime.GOOS; os {
case "darwin":
    fmt.Println("macOS")
default:
    fmt.Println("Other OS")
}
```

## 三、defer 延迟调用
`defer` 会将其后面的函数推迟到外层函数返回前执行，常用于资源释放、解锁。
```go
func read() {
    f, _ := os.Open("file.txt")
    defer f.Close() // 无论下面发生什么，函数返回前必执行
    // do something
}
```
*注意：多个 defer 执行顺序为 LIFO（后进先出）。*

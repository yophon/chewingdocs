## 系列简介

写一行 `let x = 1 + 2`,背后是 JS / Python / Lua / Ruby 共用的那层运行时——几千行代码协作把字符串变 token、变 AST、走求值或字节码 VM。**这套机制几乎每门动态语言都长一样**,亲手写一遍就等于一次性拆掉那层抽象。

本系列**不是《Crafting Interpreters》翻译**——那本书结构是教学,先把整本 Lox 一次性走完;**本系列偏"工程师视角"**:每篇对应一门语言里你天天用、却没拆过的特性,讲清楚它怎么实现、工业级怎么做、有哪些优化方向。

贯穿始终是一门玩具语言 **Mochi**(麻糬,软糯 Q 弹):动态类型、一等公民函数 + 闭包 + 类 + 模块,从 01-19 篇的树遍历版本一路推到 20-25 篇的字节码 VM,再在 26-27 篇加 GC。**宿主语言 TypeScript 为主**,关键算法附 Rust。

**完成判据**:每篇都要 Mochi 仓库有对应 commit、文章里至少一张图、「工业级是怎么做的」给出 V8 / CPython / Lua / JVM 的具体文件路径——没指针就是没看源码,不能糊弄。

**「我只想了解原理,不写代码」**:01 → 02 → 14 → 16 → 26 → 28 → 29——拿到核心心智图,知道实现层在干嘛,够你跟同事白板讨论了。

## Mochi 语法速查表

```mochi
// 不可变绑定 / 可变绑定
let x = 1
var y = 2
y = 3

// 函数(一等公民)
fn add(a, b) {
  return a + b
}

// 类(单继承,父类用 < 标注)
class Point {
  init(x, y) {
    this.x = x
    this.y = y
  }
  distance() {
    return sqrt(this.x*this.x + this.y*this.y)
  }
}
class ColorPoint < Point {
  init(x, y, color) {
    super.init(x, y)
    this.color = color
  }
}

// 控制流(强制大括号,没有"裸 if")
if x > 0 {
  print("正")
} else if x == 0 {
  print("零")
} else {
  print("负")
}

while x > 0 { x = x - 1 }
for i in 0..10 { print(i) }

// 模块
import "math"
import { sqrt, sin } from "math"
```

**字节码指令名约定(全系列统一)**:`CONST` `POP` `LOAD_LOCAL` `STORE_LOCAL` `ADD` `SUB` `MUL` `DIV` `EQ` `LT` `GT` `NOT` `JUMP` `JUMP_IF_FALSE` `CALL` `RETURN`。22 篇栈式 VM 给出完整 TS 实现。

**真值语义**:`false` `0` `""` `null` 为假,其他全为真——向 Python 看齐,但不让空容器自动为假。

**相等**:`==` 是严格相等,不做 JS 那种隐式类型转换(`"3" + 4` 直接报错)。

## 工业级源码索引

读完 Mochi,你的下一站。**每个项目给一个入口文件 + 一段说明**——按这个指针进去就能开始读。

### CPython —— 字节码 VM 的标杆,Python 自己

仓库:[`python/cpython`](https://github.com/python/cpython)

| 主题 | 文件 | 看什么 |
| --- | --- | --- |
| 字节码调度循环 | [`Python/ceval.c`](https://github.com/python/cpython/blob/main/Python/ceval.c) `_PyEval_EvalFrameDefault` | 4000 行的怪兽,computed goto + IC |
| Specializing interpreter | [`Python/generated_cases.c.h`](https://github.com/python/cpython/blob/main/Python/generated_cases.c.h) | 3.11+ 自动生成的特化指令 |
| PEG 文法 | [`Grammar/python.gram`](https://github.com/python/cpython/blob/main/Grammar/python.gram) | 3.9+ 切到 PEG 后的源文法 |
| 对象 / 类型系统 | [`Objects/typeobject.c`](https://github.com/python/cpython/blob/main/Objects/typeobject.c) | `_PyType_Lookup`(方法查找)、`super_getattro` |
| GC | [`Python/gc.c`](https://github.com/python/cpython/blob/main/Python/gc.c) | 引用计数 + 分代追踪(循环引用) |

入门切入点:**先看 `ceval.c` 里 `case TARGET(LOAD_FAST):`**,这就是 Mochi 22 篇那个 switch 的真身。

### V8 —— JavaScript 引擎,多 tier JIT 的代表

仓库:[`v8/v8`](https://github.com/v8/v8)

| 主题 | 文件 | 看什么 |
| --- | --- | --- |
| Ignition 解释器 | `src/interpreter/interpreter-generator.cc` | 用 CodeStubAssembler 生成机器码模板 |
| Sparkplug(baseline JIT) | `src/baseline/` | 2021 加的 baseline tier |
| Maglev(中端 JIT) | `src/maglev/` | 2023+ 加的中端 |
| TurboFan(顶级 JIT) | `src/compiler/pipeline.cc` | 方法 JIT 的工业标杆 |
| HiddenClass(IC 基础) | `src/objects/map.h` `src/objects/js-objects.h` | 对象 shape 系统 |
| GC(Orinoco) | `src/heap/` | 分代 + 增量 + 并发 |
| 解析器 | `src/parsing/parser.cc` | 手写递归下降 + 表达式 Pratt |

入门切入点:**`src/objects/map.h`** 看 HiddenClass 设计——读完你会理解为什么"频繁 `delete obj.x`"会让 V8 性能崩。

### Lua —— 嵌入式语言天花板,寄存器式 VM 的教科书

仓库:[`lua/lua`](https://github.com/lua/lua)

| 主题 | 文件 | 看什么 |
| --- | --- | --- |
| VM 调度循环 | `src/lvm.c` `luaV_execute` | **1000 行能读完**——栈/寄存器混合,极其干净 |
| 字节码格式 | `src/lopcodes.h` | 5-bit opcode + 操作数布局 |
| 编译器(单遍) | `src/lparser.c` `src/lcode.c` | 一遍生成字节码,不构 AST |
| GC | `src/lgc.c` | 增量 mark-sweep |
| GC API(嵌入) | `src/lapi.c` | 宿主语言怎么操作 Lua 值 |

入门切入点:**Lua 全仓库 ~25k 行 C**——是工业级语言里**最适合通读**的一个。先 `lvm.c` 再 `lparser.c`,两个文件读完你的"语言实现"心智图就稳了。**LuaJIT** 在另一个仓库 [`LuaJIT/LuaJIT`](https://github.com/LuaJIT/LuaJIT),`src/lj_record.c` 是 Tracing JIT 的起点——想看 JIT 真东西必读。

### JVM(HotSpot) —— GC 与 JIT 的工业巅峰

仓库:[`openjdk/jdk`](https://github.com/openjdk/jdk)

| 主题 | 文件 | 看什么 |
| --- | --- | --- |
| 模板解释器 | `src/hotspot/cpu/x86/templateTable_x86.cpp` | 每条字节码对应一段手写汇编 |
| C1(client JIT) | `src/hotspot/share/c1/` | 快速 JIT,linear scan 寄存器分配 |
| C2(server JIT) | `src/hotspot/share/opto/compile.cpp` | SSA + Sea-of-Nodes,**至今无对手** |
| G1 GC | `src/hotspot/share/gc/g1/` | 分代 + region + 并发 |
| ZGC | `src/hotspot/share/gc/z/` | 亚毫秒级停顿的 GC |

入门切入点:**模板解释器先看**——`templateTable_x86.cpp` 的 `iadd_template` 就是字节码 `IADD` 翻成的几条 x86 汇编,**比抽象的 IR 直观 10 倍**。

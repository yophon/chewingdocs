дёҖеҸҘиҜқе…Ҳи®°дҪҸ:гҖҢжқЎд»¶гҖҚжҳҜжҰӮзҺҮи®әйҮҢжңҖйҮҚиҰҒзҡ„дёҖдёӘеӯ—гҖӮи„ұзҰ»дәҶгҖҢеңЁд»Җд№ҲжқЎд»¶дёӢгҖҚ,д»»дҪ•жҰӮзҺҮж•°йғҪжІЎж„Ҹд№үгҖӮгҖҢ99% еҮҶзЎ®зҺҮгҖҚгҖҢеҒҮйҳізҺҮ 5%гҖҚгҖҢP еҖј 0.04гҖҚвҖ”вҖ”иҝҷдәӣж•°жң¬иә«дёҚжҳҜз»“и®ә,е…ій”®жҳҜгҖҢеңЁд»Җд№ҲеүҚжҸҗдёӢгҖҚгҖӮдёӢйқў 80% зҡ„еҸҚзӣҙи§үжЎҲдҫӢ,ж №жң¬еҺҹеӣ йғҪжҳҜгҖҢиҜ»иҖ…еҒ·еҒ·жҚўдәҶжқЎд»¶гҖҚгҖӮ

дёҖгҖҒдёәд»Җд№Ҳеҝ…йЎ»жҠҠжҰӮзҺҮеҝғжҷәе…Ҳе»әеҘҪ вҖӢ

жқҘдёүдёӘзңҹе®һеңәжҷҜ,е…Ҳж„ҹеҸ—дёҖдёӢдёҚжҮӮзҡ„д»Јд»·:

еңәжҷҜ 1:дҪ еҒҡдәҶдёӘеҲҶзұ»еҷЁ,еҮҶзЎ®зҺҮ 99%гҖӮжҹҗеӨ© PM жӢҝиҝҷдёӘеҺ»зӯӣжҹҘдёҖдёӘеҸ‘з”ҹзҺҮ 1% зҡ„дәӢд»¶(е№ҝе‘ҠзӮ№еҮ»гҖҒж¬әиҜҲгҖҒзҪ•и§Ғз–ҫз—…вҖҰвҖҰ),й—®дҪ гҖҢйў„жөӢдёәжӯЈзҡ„ж ·жң¬йҮҢ,зңҹзҡ„жңүеӨҡе°‘жҳҜжӯЈзҡ„гҖҚгҖӮзӣҙи§ү:99%гҖӮе®һйҷ…:зәҰ 50%гҖӮе·®дәҶдёҖеҖҚгҖӮ

еңәжҷҜ 2:дҪ з»ҷиҝҗиҗҘеҒҡ A/B жөӢиҜ•,з»„ A иҪ¬еҢ–зҺҮ 5.0%гҖҒз»„ B иҪ¬еҢ–зҺҮ 5.2%,P еҖј 0.04гҖӮиҝҗиҗҘе…ҙеҘӢең°е…ЁйҮҸдёҠзәҝгҖӮдёүе‘ЁеҗҺ,иҪ¬еҢ–зҺҮжІЎеҠЁгҖӮP еҖј 0.04 дёҚзӯүдәҺгҖҢB жҜ” A еҘҪзҡ„жҰӮзҺҮ 96%гҖҚгҖӮ

еңәжҷҜ 3:дҪ зңӢе…¬еҸёжҠҘе‘Ҡ,гҖҢж–°еҠҹиғҪи®©иҪ¬еҢ–зҺҮжҸҗеҚҮ 5%гҖҚ,ж•°жҚ®жҳҜзңҹзҡ„гҖӮдҪ жҢүеӯҗзҫӨзңӢ:жүӢжңәз”ЁжҲ·иҪ¬еҢ–зҺҮйҷҚдәҶгҖҒPC з”ЁжҲ·иҪ¬еҢ–зҺҮйҷҚдәҶгҖҒе№іжқҝз”ЁжҲ·иҪ¬еҢ–зҺҮд№ҹйҷҚдәҶвҖ”вҖ”е…ЁйҮҸеҚҙж¶ЁдәҶ 5%гҖӮиҝҷдёҚжҳҜ bug,иҝҷжҳҜиҫӣжҷ®жЈ®жӮ–и®әгҖӮ

еңәжҷҜеҮӯзӣҙи§үзҡ„з»“и®әе®һйҷ…жӯЈзЎ®зҡ„з»“и®ә
99% еҮҶзЎ®зҺҮзӯӣжҹҘ 1% жӮЈз—…зҺҮйҳіжҖ§еӨ§жҰӮзҺҮзңҹжӮЈз—…йҳіжҖ§дёӯзңҹжӮЈз—…еҸӘеҚ зәҰ 50%
P=0.04 жҳҫи‘—B жҜ” A еҘҪзҡ„жҰӮзҺҮ 96%гҖҢеҒҮи®ҫ A=B зҡ„дё–з•ҢйҮҢеҮәзҺ°иҝҷз§Қе·®ејӮзҡ„жҰӮзҺҮ 4%гҖҚ,е®Ңе…ЁдёҚжҳҜдёҖеӣһдәӢ
е…ЁйҮҸж¶Ё,еҲҶз»„йғҪи·Ңж•°жҚ®йҖ еҒҮиҫӣжҷ®жЈ®жӮ–и®ә,ж•°жҚ®зңҹе®һ

2.1 дёүз§Қи§ЈйҮҠ вҖӢ

  P(A) = й•ҝжңҹйҮҚеӨҚе®һйӘҢдёӯ A еҸ‘з”ҹзҡ„йў‘зҺҮжһҒйҷҗ
  дҫӢ:жү”зЎ¬еёҒ P(жӯЈйқў)=0.5,ж„ҸжҖқжҳҜжү”ж— з©·еӨҡж¬Ў,жӯЈйқўжҜ”дҫӢи¶ӢдәҺ 0.5

иҙқеҸ¶ж–Ҝжҙҫ(Bayesian)
  P(A) = жҲ‘еҜ№ A еҸ‘з”ҹзҡ„зҪ®дҝЎеәҰ,еҸҜд»ҘйҡҸиҜҒжҚ®жӣҙж–°
  дҫӢ:P(жҳҺеӨ©дёӢйӣЁ)=0.7,ж„ҸжҖқжҳҜгҖҢеҹәдәҺеҪ“еүҚжүҖзҹҘ,жҲ‘и®ӨдёәдёӢйӣЁеҸҜиғҪжҖ§ 0.7гҖҚ

дё»и§ӮзҪ®дҝЎ(Subjective/Betting)
  P(A) = жҲ‘ж„ҝж„Ҹд»ҘеӨҡе°‘иө”зҺҮдёә A жҠјжіЁ
  дҫӢ:P(A)=0.7 вҮ” жҲ‘ж„ҝж„Ҹ 7:3 жҠј A еҸ‘з”ҹ(еҗҢзӯүжңҹжңӣ)

еҗҺдёӨдёӘе…¶е®һжҳҜгҖҢеҗҢдёҖдёӘж„ҸжҖқзҡ„дёӨз§ҚиЎЁиҫҫгҖҚвҖ”вҖ”дё»и§ӮзҪ®дҝЎжҳҜиҙқеҸ¶ж–Ҝзҡ„гҖҢж“ҚдҪңжҖ§гҖҚе®ҡд№ү(з”Ёжү“иөҢиЎҢдёәж ЎеҮҶ)гҖӮ

P(A, B) = P(A | B) В· P(B)
        = P(B | A) В· P(A)        вҶҗ еҜ№з§°еҪўејҸ

P(A) = ОЈ_B P(A, B) = ОЈ_B P(A | B) В· P(B)    вҶҗ е…ЁжҰӮзҺҮе…¬ејҸ

жңҖеҗҺдёҖиЎҢеҸ«гҖҢе…ЁжҰӮзҺҮе…¬ејҸгҖҚ,ж„ҸжҖқжҳҜ:жҠҠжүҖжңүеҸҜиғҪзҡ„ B еҠ иө·жқҘ,е°ұиғҪд»ҺжқЎд»¶еӣһеҲ°иҫ№зјҳгҖӮиҝҷжҳҜиҙқеҸ¶ж–Ҝе®ҡзҗҶзҡ„еҲҶжҜҚжқҘжәҗгҖӮ

3.3 жқЎд»¶жҰӮзҺҮзҡ„еҸҚзӣҙи§ү вҖӢ

зӣҙи§үйҷ·йҳұ:P(A | B) е’Ң P(B | A) жҳҜдёҚеҗҢзҡ„гҖӮ

P(дёӢйӣЁ | еӨҡдә‘)        вү    P(еӨҡдә‘ | дёӢйӣЁ)
P(еӣҪзұҚдёӯеӣҪ | е§“еј )    вү    P(е§“еј  | еӣҪзұҚдёӯеӣҪ)
P(зҷҢз—Ү | еҢ–йӘҢйҳіжҖ§)    вү    P(еҢ–йӘҢйҳіжҖ§ | зҷҢз—Ү)

жңҖеҗҺдёҖдёӘдҫӢеӯҗжңҖиҮҙе‘ҪвҖ”вҖ”еҢ–йӘҢжҠҘе‘Ҡе‘ҠиҜүдҪ зҡ„жҳҜгҖҢеңЁдҪ жӮЈзҷҢзҡ„жқЎд»¶дёӢгҖҒеҢ–йӘҢйҳіжҖ§зҡ„жҰӮзҺҮгҖҚ(зҒөж•ҸеәҰ,йҖҡеёё 95%+),дҪ зңҹжӯЈжғізҹҘйҒ“зҡ„жҳҜгҖҢеҢ–йӘҢйҳіжҖ§зҡ„жқЎд»¶дёӢгҖҒдҪ жӮЈзҷҢзҡ„жҰӮзҺҮгҖҚгҖӮиҝҷдёӨдёӘж•°е·®ејӮжһҒеӨ§,иҙқеҸ¶ж–Ҝе®ҡзҗҶе°ұжҳҜз”ЁжқҘзҝ»иҜ‘иҝҷдёӨдёӘж–№еҗ‘зҡ„гҖӮ

еӣӣгҖҒиҙқеҸ¶ж–Ҝе®ҡзҗҶ:д»Һе·ІзҹҘеҸҚжҺЁжңӘзҹҘ вҖӢ

4.1 е…¬ејҸдёҺжңҜиҜӯ вҖӢ

                P(B | A) В· P(A)
P(A | B) = в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
                    P(B)

         = P(B | A) В· P(A) / [ P(B|A)В·P(A) + P(B|В¬A)В·P(В¬A) ]

жҜҸдёӘйғЁеҲҶйғҪжңүеҗҚеӯ—,и®°дҪҸеҗҚеӯ—жҜ”и®°дҪҸе…¬ејҸйҮҚиҰҒ:

йғЁеҲҶеҗҚеӯ—еҗ«д№ү
P(A)е…ҲйӘҢ(prior)жІЎзңӢеҲ° B д№ӢеүҚ,дҪ еҜ№ A зҡ„зҪ®дҝЎеәҰ
`P(BA)`дјјз„¶(likelihood)
`P(AB)`еҗҺйӘҢ(posterior)
P(B)иҜҒжҚ®(evidence)B ж•ҙдҪ“еҸ‘з”ҹзҡ„жҰӮзҺҮ,иө·еҪ’дёҖеҢ–дҪңз”Ё

иҙқеҸ¶ж–Ҝзҡ„ж ёеҝғ:

еҗҺйӘҢ вҲқ дјјз„¶ Г— е…ҲйӘҢ

ж ёеҝғзӣҙи§ү:иҙқеҸ¶ж–Ҝе®ҡзҗҶе°ұжҳҜгҖҢж•°жҚ®жӣҙж–°дҝЎеҝөгҖҚзҡ„ж•°еӯҰеҢ–гҖӮе…ҲйӘҢжҳҜдҪ зҡ„еҲқе§ӢзҢңжөӢгҖҒдјјз„¶жҳҜгҖҢеҒҮеҰӮзҢңеҜ№дәҶ,и§ӮжөӢеҲ°еҪ“еүҚж•°жҚ®зҡ„еҸҜиғҪжҖ§гҖҚгҖҒеҗҺйӘҢжҳҜжҠҠдёӨиҖ…з»“еҗҲеҗҺзҡ„ж–°зҢңжөӢгҖӮжүҖжңү RAGгҖҒrerankingгҖҒжҺЁиҚҗгҖҒйЈҺжҺ§гҖҒеһғеңҫиҝҮж»Өзҡ„ж ёеҝғиҝҗиҪ¬ж–№ејҸйғҪжҳҜиҝҷеҘ—гҖӮ

4.2 еҹәзЎҖзҺҮи°¬иҜҜ:99% еҮҶзЎ®зҺҮзҡ„зңҹзӣё вҖӢ

еӣһеҲ°ејҖеӨҙзҡ„дҫӢеӯҗгҖӮжҹҗеҢ»еӯҰеҢ–йӘҢ:

  • зҒөж•ҸеәҰ(зңҹжӮЈз—… вҶ’ йҳіжҖ§): 99%
  • зү№ејӮеәҰ(жІЎжӮЈз—… вҶ’ йҳҙжҖ§): 99%
  • дәәзҫӨжӮЈз—…зҺҮ: 1%

й—®йўҳ:дҪ еҢ–йӘҢйҳіжҖ§,зңҹжӯЈжӮЈз—…зҡ„жҰӮзҺҮжҳҜеӨҡе°‘?

зӣҙи§ү:99%гҖӮжӯЈзЎ®зӯ”жЎҲ:50%гҖӮи®©иҙқеҸ¶ж–Ҝз®—жё…жҘҡ:

A = зңҹжӮЈз—…
B = еҢ–йӘҢйҳіжҖ§

P(A)       = 0.01           вҶҗ е…ҲйӘҢ,дәәзҫӨжӮЈз—…зҺҮ
P(В¬A)      = 0.99
P(B | A)   = 0.99           вҶҗ зҒөж•ҸеәҰ
P(B | В¬A)  = 0.01           вҶҗ 1 - зү№ејӮеәҰ,еҒҮйҳізҺҮ

P(B) = P(B|A)В·P(A) + P(B|В¬A)В·P(В¬A)
     = 0.99 В· 0.01 + 0.01 В· 0.99
     = 0.0099 + 0.0099
     = 0.0198

P(A | B) = 0.99 В· 0.01 / 0.0198 = 0.5

зӣҙи§үй”ҷеңЁе“Ә:99% еҮҶзЎ®зҺҮиҜҙзҡ„жҳҜгҖҢжқЎд»¶гҖҚжҰӮзҺҮвҖ”вҖ”е·ІзҹҘжӮЈз—…ж—¶йҳіжҖ§зҡ„жҰӮзҺҮ;иҜ»иҖ…еҒ·еҒ·жҠҠе®ғеҪ“жҲҗгҖҢйҳіжҖ§ж—¶жӮЈз—…зҡ„жҰӮзҺҮгҖҚ,жқЎд»¶жҗһеҸҚдәҶгҖӮ

NumPy йӘҢиҜҒ:

python
import numpy as np

N = 1_000_000
sick     = np.random.rand(N) < 0.01   # 1% жӮЈз—…
positive = np.where(
    sick,
    np.random.rand(N) < 0.99,         # жӮЈз—…иҖ… 99% йҳіжҖ§
    np.random.rand(N) < 0.01,         # еҒҘеә·иҖ… 1% йҳіжҖ§(еҒҮйҳі)
)

# йҳіжҖ§дёӯзңҹжӮЈз—…зҡ„жҜ”дҫӢ
print((sick & positive).sum() / positive.sum())
# вүҲ 0.5

йҒҝеқ‘:д»»дҪ•гҖҢеҲҶзұ»еҷЁеҮҶзЎ®зҺҮ 99%гҖҚзұ»йҷҲиҝ°,е…Ҳй—®еҹәзЎҖзҺҮ(base rate)гҖӮеҹәзЎҖзҺҮи¶ҠдҪҺ,гҖҢйҳіжҖ§зҡ„зңҹйҳізҺҮгҖҚи¶ҠжҺҘиҝ‘ 0гҖӮиҝҷе°ұжҳҜдёәд»Җд№ҲеҢ»еӯҰзӯӣжҹҘгҖҒж¬әиҜҲжЈҖжөӢгҖҒеһғеңҫиҝҮж»ӨйғҪдёҚиғҪеҸӘзңӢеҮҶзЎ®зҺҮ,еҝ…йЎ»зңӢ precision/recall/PR жӣІзәҝгҖӮ

4.3 е·ҘзЁӢеҜ№еә”:иҙқеҸ¶ж–ҜеһғеңҫйӮ®д»¶иҝҮж»Ө вҖӢ

з»Ҹе…ёзҡ„ Paul Graham еһғеңҫйӮ®д»¶иҝҮж»Өе°ұжҳҜиҙқеҸ¶ж–Ҝе®ҡзҗҶ:

P(еһғеңҫ | йӮ®д»¶еҗ«"viagra"зӯүе…ій”®иҜҚ) 
  = P(еҗ«е…ій”®иҜҚ | еһғеңҫ) В· P(еһғеңҫ) / P(еҗ«е…ій”®иҜҚ)

жҜҸжқҘдёҖе°ҒйӮ®д»¶,з”ЁеҺҶеҸІж ·жң¬з»ҹи®Ў P(жҜҸдёӘиҜҚ | еһғеңҫ) е’Ң P(жҜҸдёӘиҜҚ | йқһеһғеңҫ),еҒҮи®ҫиҜҚд№Ӣй—ҙзӢ¬з«Ӣ(naive Bayes,naive е°ұ naive еңЁиҝҷйҮҢ),жҠҠжүҖжңүиҜҚд№ҳиө·жқҘ:

P(еһғеңҫ | ж•ҙе°ҒйӮ®д»¶) вҲқ P(еһғеңҫ) В· вҲҸ P(иҜҚ i | еһғеңҫ)

з®ҖеҚ•зІ—жҡҙ,дҪҶеңЁиҝҮж»Өд»»еҠЎдёҠй•ҝжңҹзўҫеҺӢеӨҚжқӮжЁЎеһӢвҖ”вҖ”еӣ дёәе…ҲйӘҢйқһеёёзЁіе®ҡгҖҒзү№еҫҒзӢ¬з«ӢжҖ§гҖҢеҒҮй”ҷд№ҹй”ҷеҫ—еҜ№гҖҚгҖҒи®ӯз»ғж ·жң¬е……и¶ігҖӮ

дә”гҖҒзӢ¬з«ӢгҖҒдёҚзӣёе…ігҖҒдә’ж–Ҙ:дёүдёӘеёёиў«ж··зҡ„жҰӮеҝө вҖӢ

жҰӮеҝөж•°еӯҰе®ҡд№үзӣҙи§ү
зӢ¬з«ӢP(A, B) = P(A) В· P(B) еҚі `P(AB) = P(A)`
дёҚзӣёе…іCov(X, Y) = 0 еҚі E[XY] = E[X]E[Y]X е’Ң Y зҡ„гҖҢзәҝжҖ§е…ізі»гҖҚдёәйӣ¶
дә’ж–ҘA вҲ© B = вҲ… еҚі P(A, B) = 0A е’Ң B дёҚеҸҜиғҪеҗҢж—¶еҸ‘з”ҹ

е®№жҳ“ж··зҡ„ең°ж–№:

зӢ¬з«Ӣ вҮ’ дёҚзӣёе…і (еҸҚиҝҮжқҘдёҚжҲҗз«Ӣ)
зӢ¬з«Ӣ вҮ’ дёҚдә’ж–Ҙ (дә’ж–Ҙж„Ҹе‘ізқҖгҖҢB еҸ‘з”ҹдәҶ A дёҖе®ҡдёҚеҸ‘з”ҹгҖҚ,йқһеёёејәзҡ„дҫқиө–)
дә’ж–Ҙ вҮ’ дёҚзӢ¬з«Ӣ (йҷӨйқһиҮіе°‘дёҖдёӘзҡ„жҰӮзҺҮжҳҜ 0)

5.1 зӢ¬з«Ӣ вү  дёҚзӣёе…і:з»Ҹе…ёеҸҚдҫӢ вҖӢ

X жңҚд»Һ [-1, 1] еқҮеҢҖеҲҶеёғ
Y = XВІ

E[X] = 0
E[XY] = E[XВі] = 0   (еҘҮеҮҪж•°)
Cov(X, Y) = E[XY] - E[X]E[Y] = 0

жүҖд»Ҙ X е’Ң Y дёҚзӣёе…ігҖӮдҪҶ Y е®Ңе…Ёз”ұ X еҶіе®ҡ,**з»қеҜ№дёҚзӢ¬з«Ӣ**гҖӮ

гҖҢзӣёе…іжҖ§гҖҚеҸӘиғҪжҚ•жҚүзәҝжҖ§е…ізі»вҖ”вҖ”иҝҷе°ұжҳҜдёәд»Җд№Ҳ ML йҮҢиҰҒз”ЁгҖҢдә’дҝЎжҒҜгҖҚ(mutual information)жқҘеәҰйҮҸйқһзәҝжҖ§дҫқиө–гҖӮ

5.2 дә’ж–Ҙ вҮҸ зӢ¬з«Ӣ вҖӢ

жҠӣдёҖж¬ЎзЎ¬еёҒ:
A = жӯЈйқў
B = еҸҚйқў

A е’Ң B дә’ж–Ҙ(P(A,B)=0)
дҪҶ P(A|B) = 0 вү  P(A) = 0.5
жүҖд»Ҙ**дёҚзӢ¬з«Ӣ**вҖ”вҖ”B еҸ‘з”ҹејәзғҲеҪұе“ҚдәҶ A зҡ„жҰӮзҺҮ

з»ҸйӘҢжі•еҲҷ:гҖҢзӢ¬з«ӢгҖҚжҳҜжңҖејәзҡ„жқЎд»¶,гҖҢдёҚзӣёе…ігҖҚжҳҜдёӯзӯүејәеәҰ,гҖҢдә’ж–ҘгҖҚе…¶е®һжҳҜдёҖз§Қзү№ж®Ҡзҡ„гҖҢжһҒејәдҫқиө–гҖҚгҖӮи®әж–ҮйҮҢгҖҢassume features are independentгҖҚжҳҜйқһеёёзЎ¬зҡ„еҒҮи®ҫ,naive Bayes д№ӢжүҖд»ҘгҖҢnaiveгҖҚе°ұжҳҜеӣ дёәе®ғеҒҮи®ҫзү№еҫҒзӢ¬з«Ӣ(еҫҖеҫҖдёҚжҲҗз«Ӣ,дҪҶеӨҹз”Ё)гҖӮ

е…ӯгҖҒMonty Hall дёүй—Ёй—®йўҳ:жқЎд»¶жҰӮзҺҮжңҖз»Ҹе…ёжЎҲдҫӢ вҖӢ

6.1 йўҳзӣ® вҖӢ

дёүжүҮй—Ё,дёҖжүҮеҗҺйқўжңүиҪҰ,дёӨжүҮеҗҺйқўжҳҜзҫҠгҖӮдҪ йҖүдәҶ 1 еҸ·й—Ё;дё»жҢҒдәә(зҹҘйҒ“иҪҰеңЁе“Ә)д»Һеү©дёӢдёӨжүҮйҮҢжү“ејҖдёҖжүҮжңүзҫҠзҡ„й—Ё(жҜ”еҰӮ 3 еҸ·);й—®йўҳ:дҪ иҰҒдёҚиҰҒжҚўеҲ° 2 еҸ·й—Ё?

еҮӯзӣҙи§ү:еү©дёӨжүҮ,еҗ„ 1/2 жҰӮзҺҮ,жҚўдёҚжҚўйғҪиЎҢгҖӮ

жӯЈзЎ®зӯ”жЎҲ:жҚў!жҚўзҡ„иҜқдёӯиҪҰжҰӮзҺҮ 2/3,дёҚжҚўеҸӘжңү 1/3гҖӮ

6.2 з”ЁжқЎд»¶жҰӮзҺҮз®—жё…жҘҡ вҖӢ

дәӢд»¶е®ҡд№ү:
  C_i:иҪҰеңЁз¬¬ i еҸ·й—ЁеҗҺйқў     (i=1,2,3)
  H_i:дё»жҢҒдәәжү“ејҖз¬¬ i еҸ·й—Ё
  дҪ жңҖеҲқйҖүзҡ„жҳҜ 1 еҸ·й—ЁгҖӮ

е…ҲйӘҢ:P(C_1) = P(C_2) = P(C_3) = 1/3

и§ӮеҜҹеҲ°зҡ„жқЎд»¶:дё»жҢҒдәәжү“ејҖдәҶ 3 еҸ·й—Ё(H_3)гҖӮ

жұӮ:P(C_2 | H_3) vs P(C_1 | H_3)

дё»жҢҒдәәиЎҢдёәи§„еҲҷ(е…ій”®):
  иӢҘ C_1(иҪҰеңЁ 1):дё»жҢҒдәәйҡҸжңәд»Һ 2гҖҒ3 йҖүдёҖжүҮ,жүҖд»Ҙ P(H_3 | C_1) = 1/2
  иӢҘ C_2(иҪҰеңЁ 2):дё»жҢҒдәәеҸӘиғҪејҖ 3(еӣ дёәдёҚиғҪејҖ 2,д№ҹдёҚиғҪејҖдҪ йҖүзҡ„ 1),жүҖд»Ҙ P(H_3 | C_2) = 1
  иӢҘ C_3(иҪҰеңЁ 3):дё»жҢҒдәәдёҚиғҪејҖ 3,жүҖд»Ҙ P(H_3 | C_3) = 0

иҙқеҸ¶ж–Ҝ:
  P(H_3) = P(H_3|C_1)В·P(C_1) + P(H_3|C_2)В·P(C_2) + P(H_3|C_3)В·P(C_3)
         = 1/2 В· 1/3 + 1 В· 1/3 + 0
         = 1/2

  P(C_1 | H_3) = P(H_3|C_1)В·P(C_1) / P(H_3) = (1/2 В· 1/3) / (1/2) = 1/3
  P(C_2 | H_3) = P(H_3|C_2)В·P(C_2) / P(H_3) = (1 В· 1/3) / (1/2)   = 2/3

жүҖд»ҘжҚўжҜ”дёҚжҚўеҘҪ,жҰӮзҺҮд»Һ 1/3 ж¶ЁеҲ° 2/3гҖӮ

6.3 зӣҙи§үи§ЈйҮҠ вҖӢ

ж ёеҝғзӣҙи§ү:дҪ жңҖеҲқйҖү 1 еҸ·й—Ёж—¶,жңү 1/3 жҰӮзҺҮйҖүдёӯгҖҒ2/3 жҰӮзҺҮйҖүй”ҷгҖӮдё»жҢҒдәәејҖй—ЁжҳҜеёҰдҝЎжҒҜзҡ„вҖ”вҖ”д»–зҹҘйҒ“иҪҰеңЁе“Ә,д»–еҝ…з„¶ејҖдёҖжүҮжңүзҫҠзҡ„й—ЁгҖҒдёҚејҖиҪҰйӮЈжүҮгҖӮжүҖд»ҘгҖҢдё»жҢҒдәәејҖй—ЁгҖҚзҡ„еҠЁдҪңжҠҠ 2/3 зҡ„гҖҢдёҚеңЁ 1 еҸ·гҖҚжҰӮзҺҮе…ЁйғЁйӣҶдёӯеҲ°дәҶгҖҢ2 еҸ·гҖҚдёҠ(еӣ дёәд»–её®дҪ жҺ’йҷӨдәҶ 3 еҸ·)гҖӮ

def play(switch):
    car = np.random.randint(3)
    pick = 0   # дёҚеӨұдёҖиҲ¬жҖ§,жҖ»жҳҜе…ҲйҖү 0 еҸ·
    # дё»жҢҒдәәејҖдёҖжүҮ:дёҚжҳҜиҪҰгҖҒдёҚжҳҜдҪ йҖүзҡ„
    options = [d for d in range(3) if d != pick and d != car]
    revealed = np.random.choice(options)
    if switch:
        # жҚўеҲ°еү©дёӢйӮЈжүҮ
        new_pick = [d for d in range(3) if d != pick and d != revealed][0]
        return new_pick == car
    return pick == car

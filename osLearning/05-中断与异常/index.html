й«ҳ PPS:   з¬¬дёҖдёӘдёӯж–ӯеҗҺ,е…ідёӯж–ӯж”№иҪ®иҜў,еӨ„зҗҶе®ҢдёҖжү№еҶҚејҖдёӯж–ӯ

    йҒҝе…Қй«ҳ PPS ж—¶дёӯж–ӯйЈҺжҡҙ вҖ”вҖ” жҳҜзҺ°д»ЈзҪ‘еҚЎй©ұеҠЁзҡ„ж Үй…ҚгҖӮ

    дёғгҖҒејӮеёё:CPU иҮӘе·ұеҮәй”ҷ

    7.1 дёүзұ»ејӮеёё

    зјәйЎө(Page Fault):и®ҝй—®зҡ„йЎөдёҚеңЁзү©зҗҶеҶ…еӯҳ вҶ’ еҶ…ж ёжҚўе…Ҙ вҶ’ йҮҚж–°жү§иЎҢиҜҘжҢҮд»Ө

    7.2 зјәйЎөдёӯж–ӯ(Page Fault):жңҖйў‘з№Ғзҡ„ејӮеёё

    char *p = mmap(...);    // mmap еҸӘжҳҜеҲҶй…ҚиҷҡжӢҹең°еқҖ,жІЎзңҹеҲҶзү©зҗҶйЎө
p[0] = 1;               // и®ҝй—®и§ҰеҸ‘зјәйЎө
                        // еҶ…ж ёеҲҶй…Қзү©зҗҶйЎө,е»әз«Ӣжҳ е°„,з»§з»ӯжү§иЎҢ

    иҜҰи§Ғ 06 зҜҮиҷҡжӢҹеҶ…еӯҳ вҖ”вҖ” зјәйЎөжҳҜ OS еҶ…еӯҳз®ЎзҗҶзҡ„ж ёеҝғжңәеҲ¶гҖӮ

    • иҝӣзЁӢжҚўж ё вҶ’ L1/L2 cache е…ЁеӨұж•Ҳ
    • дёӯж–ӯеңЁдёҚеҗҢж ёеӨ„зҗҶ вҶ’ зҪ‘з»ңеҢ…еҲ°еӨ„д№ұи·‘

    й«ҳжҖ§иғҪеңәжҷҜеёёи§Ғжһ¶жһ„:

    зҰҒеҝҢ:й•ҝж—¶й—ҙзҰҒдёӯж–ӯ = й”ҷиҝҮдёӯж–ӯ = зі»з»ҹеҚЎйЎҝгҖӮ

    9.2 е®һж—¶зі»з»ҹзҡ„йңҖжұӮ

    е®һж—¶(real-time)зі»з»ҹиҰҒжұӮе“Қеә”ж—¶й—ҙзЎ®е®ҡжҖ§ вҖ”вҖ” е…ій”®ж“ҚдҪңжңҹй—ҙдёҚиғҪиў«дёӯж–ӯжү“ж–ӯгҖӮ

    Linux дё»зәҝдёҚжҳҜе®һж—¶еҶ…ж ё,дҪҶжңү PREEMPT_RT иЎҘдёҒ и®©е®ғжҺҘиҝ‘е®һж—¶гҖӮ

    еҚҒгҖҒsyscall д№ҹжҳҜдёҖз§Қ"дёӯж–ӯ"

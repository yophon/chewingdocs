# Flutter 音视频

Flutter 自身不内置音视频,通过 plugin 调底层(Android ExoPlayer / iOS AVPlayer / Web HTMLVideoElement)。

四个常见场景:

| 场景 | 推荐 |
| --- | --- |
| 播放视频(本地 / 网络) | `video_player` + `chewie` |
| 拍照 / 录像 | `camera` |
| 播放音频 | `just_audio`(推荐) / `audioplayers` |
| 录音 | `record` |
| 直播(RTMP / WebRTC) | `agora_rtc_engine` / `tencent_cloud_chat` |

---

## 一、video_player:基础视频播放

官方包,功能基础但稳定。

```yaml
dependencies:
  video_player: ^2.9.2
```

### iOS / Android 配置

`ios/Podfile` 加:

```ruby
platform :ios, '12.0'   # video_player 要求
```

`android/app/build.gradle`:

```text
minSdkVersion 21
```

网络视频要 HTTPS。HTTP 需配置 ATS 例外(iOS)或 `usesCleartextTraffic`(Android)。

### 基本用法

```dart
class VideoDemo extends StatefulWidget {
  @override State<VideoDemo> createState() => _S();
}

class _S extends State<VideoDemo> {
  late VideoPlayerController _ctrl;

  @override
  void initState() {
    super.initState();
    _ctrl = VideoPlayerController.networkUrl(
      Uri.parse('https://flutter.github.io/assets-for-api-docs/assets/videos/butterfly.mp4'),
    )..initialize().then((_) => setState(() {}));
  }

  @override
  void dispose() {
    _ctrl.dispose();             // ⚠️ 必须释放
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    if (!_ctrl.value.isInitialized) return const CircularProgressIndicator();

    return AspectRatio(
      aspectRatio: _ctrl.value.aspectRatio,
      child: Stack(
        children: [
          VideoPlayer(_ctrl),
          VideoProgressIndicator(_ctrl, allowScrubbing: true),
        ],
      ),
    );
  }
}
```

### 控制

```dart
_ctrl.play();
_ctrl.pause();
_ctrl.seekTo(const Duration(seconds: 10));
_ctrl.setVolume(0.5);
_ctrl.setPlaybackSpeed(1.5);
_ctrl.setLooping(true);

_ctrl.addListener(() {
  print(_ctrl.value.position);   // 当前位置
  print(_ctrl.value.isPlaying);
});
```

### 来源类型

```dart
VideoPlayerController.asset('assets/v.mp4')           // 资源
VideoPlayerController.networkUrl(Uri.parse('...'))    // 网络
VideoPlayerController.file(File('/path/to.mp4'))      // 本地文件
VideoPlayerController.contentUri(uri)                 // Android Content URI
```

## 二、chewie:更好看的播放器 UI

video_player 不带控件,自己写麻烦。`chewie` 包了一套通用 UI:

```yaml
dependencies:
  video_player: ^2.9.2
  chewie: ^1.8.5
```

```dart
late ChewieController _chewieCtrl;

@override
void initState() {
  super.initState();
  _ctrl = VideoPlayerController.networkUrl(Uri.parse('...'));
  _chewieCtrl = ChewieController(
    videoPlayerController: _ctrl,
    autoPlay: true,
    looping: false,
    aspectRatio: 16 / 9,
    allowFullScreen: true,
    placeholder: Container(color: Colors.black),
    materialProgressColors: ChewieProgressColors(
      playedColor: Colors.red,
      handleColor: Colors.red,
    ),
  );
}

@override
void dispose() {
  _ctrl.dispose();
  _chewieCtrl.dispose();
  super.dispose();
}

@override
Widget build(BuildContext context) {
  return Chewie(controller: _chewieCtrl);
}
```

自带:播放暂停、进度条、全屏、倍速、字幕等。

## 四、camera:拍照与录像

```yaml
dependencies:
  camera: ^0.11.0+2
```

### 权限

iOS `Info.plist`:

```xml
<key>NSCameraUsageDescription</key>
<string>用于拍摄商品照片</string>
<key>NSMicrophoneUsageDescription</key>
<string>用于录像收音</string>
```

Android `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
```

### 用法

```dart
class CameraDemo extends StatefulWidget {
  @override State<CameraDemo> createState() => _S();
}

class _S extends State<CameraDemo> {
  CameraController? _ctrl;
  late List<CameraDescription> _cameras;

  @override
  void initState() {
    super.initState();
    _setup();
  }

  Future<void> _setup() async {
    _cameras = await availableCameras();
    _ctrl = CameraController(
      _cameras.first,                  // 默认后置
      ResolutionPreset.medium,
      enableAudio: true,
    );
    await _ctrl!.initialize();
    if (mounted) setState(() {});
  }

  @override
  void dispose() {
    _ctrl?.dispose();
    super.dispose();
  }

  Future<void> _takePicture() async {
    final file = await _ctrl!.takePicture();
    print('保存到:${file.path}');
  }

  Future<void> _toggleCamera() async {
    final cur = _ctrl!.description;
    final next = _cameras.firstWhere((c) => c.lensDirection != cur.lensDirection);
    await _ctrl!.setDescription(next);
  }

  @override
  Widget build(BuildContext context) {
    if (_ctrl == null || !_ctrl!.value.isInitialized) {
      return const Center(child: CircularProgressIndicator());
    }
    return Stack(
      children: [
        CameraPreview(_ctrl!),
        Positioned(
          bottom: 30, left: 0, right: 0,
          child: Center(
            child: FloatingActionButton(
              onPressed: _takePicture,
              child: const Icon(Icons.camera),
            ),
          ),
        ),
      ],
    );
  }
}
```

### 录像

```dart
await _ctrl!.startVideoRecording();
// ...一段时间后
final file = await _ctrl!.stopVideoRecording();
print(file.path);
```

### 实时帧分析(扫码、人脸识别)

```dart
_ctrl!.startImageStream((image) {
  // image: CameraImage,YUV / BGRA 格式
  // 把 bytes 喂给 ML Kit / OpenCV
});

_ctrl!.stopImageStream();
```

## 六、just_audio:音频播放(推荐)

```yaml
dependencies:
  just_audio: ^0.9.40
```

```dart
final player = AudioPlayer();

await player.setUrl('https://example.com/song.mp3');
await player.setAsset('assets/song.mp3');
await player.setFilePath('/local/path/song.mp3');

player.play();
player.pause();
player.stop();
player.seek(const Duration(seconds: 30));
player.setVolume(0.5);
player.setSpeed(1.5);

// 监听
player.positionStream.listen((d) => print('当前 $d'));
player.durationStream.listen((d) => print('总时长 $d'));
player.playerStateStream.listen((s) {
  print(s.playing);
  print(s.processingState);   // loading / buffering / ready / completed
});

// 释放
player.dispose();
```

### 播放列表

```dart
final playlist = ConcatenatingAudioSource(children: [
  AudioSource.uri(Uri.parse('https://.../1.mp3')),
  AudioSource.uri(Uri.parse('https://.../2.mp3')),
  AudioSource.asset('assets/local.mp3'),
]);

await player.setAudioSource(playlist);
await player.seekToNext();
await player.seekToPrevious();
```

## 八、直播 / 实时通信(WebRTC)

四个主流方案:

| 库 | 来源 | 特点 |
| --- | --- | --- |
| `agora_rtc_engine` | Agora 声网 | 海外强、文档好 |
| `tencent_cloud_chat` / `live_room_uikit_flutter` | 腾讯云 | 国内主流 |
| `zego_express_engine` | 即构 | 国内主流 |
| `flutter_webrtc` | 开源 | 自己搭信令服务 |

商用直播首选**云厂商 SDK**,免维护信令、转码、CDN。

### Agora 示例

```yaml
dependencies:
  agora_rtc_engine: ^6.5.1
```

```dart
final engine = createAgoraRtcEngine();
await engine.initialize(const RtcEngineContext(appId: 'YOUR_APP_ID'));
await engine.enableVideo();
await engine.startPreview();

// 加入房间
await engine.joinChannel(
  token: 'token',
  channelId: 'room1',
  uid: 0,
  options: const ChannelMediaOptions(),
);

engine.registerEventHandler(RtcEngineEventHandler(
  onUserJoined: (_, uid, __) => print('$uid 加入'),
  onUserOffline: (_, uid, __) => print('$uid 离开'),
));

// 离开
await engine.leaveChannel();
await engine.release();
```

UI 部分用 `AgoraVideoView` 渲染本地 / 远端画面。

### 推流(RTMP)

```dart
await engine.startRtmpStreamWithoutTranscoding('rtmp://...');
```

观看端用 video_player + HLS 或拉流地址即可。

### 直播间常见 UI

- 主播头像 + 关注按钮
- 弹幕滚屏(用 `flutter_marquee` 或自定义)
- 礼物动画(Lottie / Rive,回顾 15)
- 在线人数 / 点赞数(WebSocket 推送)

这些是 UI 层,跟具体直播 SDK 无关。

## 十、常见坑

### 1. 忘记 dispose

`VideoPlayerController` / `AudioPlayer` / `CameraController` 不释放 → 内存泄漏 + 后台仍在播放(回顾 06)。

### 2. iOS 需要 HTTPS

HTTP 视频 / 音频在 iOS 默认被 ATS 拦。要么换 HTTPS,要么 Info.plist 加例外(审核会问理由)。

### 3. Android 后台被杀

后台播放需要 foreground service。`audio_service` 帮你处理。

### 4. 视频窗口比例错乱

```dart
// ✅ 用 video 自带的比例
AspectRatio(
  aspectRatio: ctrl.value.aspectRatio,
  child: VideoPlayer(ctrl),
)
```

### 5. camera 切换镜头黑屏

要先 `pause` → `setDescription` → `resume`,或重建 Controller。

### 6. 多视频同时播放卡顿

iOS 同时播 3 个以上 H264 视频会很卡(硬件解码器有限)。
列表里建议一次只播一个,其他 paused 或用 GIF/WebP 占位。

### 7. 录像文件过大

iOS 默认录 1080p 30fps 一分钟约 100MB。**手动设 ResolutionPreset / 编码参数**或后期压缩(用 `video_compress`)。

## 十一、性能 tips

- **播放器列表**:每个 item 单独 Controller 太重,用一个公用 Controller,切换 source
- **首帧优化**:网络视频显示封面图(`placeholder`),video init 完后切到 VideoPlayer
- **预加载**:页面 push 之前 init 下一个视频(类似抖音双 buffer)
- **图片处理**:`image` 包做缩放,然后再写文件
- **camera 截图**:`takePicture` 比 `imageStream + 编码` 快得多

## 十四、心智模型

```
播放视频  → video_player + chewie    (90% 场景)
拍照选图  → image_picker             (90% 场景)
录像/复杂 → camera                   (扫码、滤镜)
播音乐    → just_audio + audio_service
录音      → record
直播      → 云厂商 SDK(国内 Tencent/ZEGO,海外 Agora)
```

**核心**:**优先用现成包,不要重复造轮子**。音视频原生开发坑深,Flutter 的 plugin 已经填了 90% 平台差异。

如果发现现成包不满足,**做 Plugin 包一层原生 ExoPlayer / AVPlayer**(回顾 16),而不是绕过它。

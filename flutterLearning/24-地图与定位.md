# Flutter 地图与定位

地图集成是 Flutter 项目里最容易踩坑的领域:**国内 / 国外不同 SDK、坐标系不同、权限要求不同**。

按"用户在哪、目标用户是谁"决定选哪个。

---

## 一、选型决策树

```
目标市场?
├─ 海外(出海)
│    └─ Google Maps  →  google_maps_flutter
│         + Apple MapKit(iOS 原生)
│
└─ 国内
     ├─ 高德           →  amap_flutter_map
     ├─ 百度           →  flutter_baidu_mapapi_map
     ├─ 腾讯地图       →  flutter_tencentmap
     └─ 全平台兼容     →  flutter_map(开源,基于瓦片)
```

国内项目:Google Maps **不能用**(没墙内服务),数据返回是空白。

---

## 二、坐标系陷阱(最重要的国内常识)

中国大陆有三套坐标系,**绝对不能混用**:

| 名称 | 全称 | 谁用 |
| --- | --- | --- |
| **WGS84** | 国际标准(GPS 原始) | Google、Apple、海外 |
| **GCJ-02** | 火星坐标(国测局加密) | 高德、腾讯、Google 中国 |
| **BD-09** | 百度坐标(GCJ-02 二次加密) | 百度地图 |

### 后果

把 WGS84 直接画到高德,**位置会偏移 50~500 米**。

### 转换

```dart
import 'package:lat_lng_converter/lat_lng_converter.dart';

final wgs = LatLng(31.2304, 121.4737);
final gcj = LatLngConverter.wgs84ToGcj02(wgs);
final bd  = LatLngConverter.gcj02ToBd09(gcj);
```

或后端转好返回。**所有地图 SDK 拿到的"原始定位"是 WGS84,显示到对应地图时必须转**。

## 三、google_maps_flutter:海外首选

```yaml
dependencies:
  google_maps_flutter: ^2.9.0
```

### 申请 API Key

Google Cloud Console → 启用 Maps SDK for iOS / Android → 创建 API Key。

iOS `AppDelegate.swift`:

```swift
import GoogleMaps

@UIApplicationMain
@objc class AppDelegate: FlutterAppDelegate {
  override func application(...) -> Bool {
    GMSServices.provideAPIKey("YOUR_KEY")
    GeneratedPluginRegistrant.register(with: self)
    return super.application(...)
  }
}
```

Android `AndroidManifest.xml`:

```xml
<application>
  <meta-data
    android:name="com.google.android.geo.API_KEY"
    android:value="YOUR_KEY" />
</application>
```

### 用法

```dart
GoogleMap(
  initialCameraPosition: CameraPosition(
    target: LatLng(37.7749, -122.4194),
    zoom: 12,
  ),
  onMapCreated: (controller) => _ctrl.complete(controller),
  markers: {
    Marker(
      markerId: const MarkerId('1'),
      position: const LatLng(37.7749, -122.4194),
      infoWindow: const InfoWindow(title: 'San Francisco'),
    ),
  },
  polylines: {
    Polyline(
      polylineId: const PolylineId('p1'),
      points: const [LatLng(37.77, -122.41), LatLng(37.78, -122.42)],
      color: Colors.blue,
      width: 5,
    ),
  },
  myLocationEnabled: true,
  myLocationButtonEnabled: true,
)
```

### 移动相机

```dart
final ctrl = await _ctrl.future;
ctrl.animateCamera(CameraUpdate.newLatLngZoom(LatLng(31.2, 121.5), 14));
```

### 注意

- iOS Maps 性能比 Android 好
- 离线下载需 Google Places(收费)
- 流量收费:每月 $200 免费额度,超量按地图加载次数计

## 四、高德地图(国内主流)

```yaml
dependencies:
  amap_flutter_map: ^3.0.0
  amap_flutter_location: ^3.0.0
```

### 配置 Key

`高德开放平台 → 我的应用 → 创建`,获取 iOS / Android Key。

iOS `AppDelegate.swift` 或 main.dart:

```dart
AMapFlutterMap.setApiKey(
  AMapApiKey(iosKey: 'IOS_KEY', androidKey: 'ANDROID_KEY'),
);
AMapFlutterMap.updatePrivacyShow(true, true);
AMapFlutterMap.updatePrivacyAgree(true);    // 隐私合规
```

⚠️ **国内合规要求**:必须先用户同意隐私后,才能调 `updatePrivacyAgree(true)`。否则高德 SDK 不工作。

### 用法

```dart
AMapWidget(
  privacyStatement: const AMapPrivacyStatement(
    hasContains: true, hasShow: true, hasAgree: true,
  ),
  initialCameraPosition: const CameraPosition(
    target: LatLng(39.909, 116.397),
    zoom: 13,
  ),
  markers: {
    Marker(
      position: const LatLng(39.909, 116.397),
      infoWindow: const InfoWindow(title: '天安门'),
    ),
  },
  onMapCreated: (controller) {
    _ctrl = controller;
  },
  myLocationStyleOptions: MyLocationStyleOptions(true),
)
```

## 六、flutter_map:开源 / 跨平台一致

不依赖任何商业地图 SDK,**用瓦片(tile)拼图**。可以用 OpenStreetMap、Mapbox、自建瓦片服务。

```yaml
dependencies:
  flutter_map: ^7.0.2
  latlong2: ^0.9.1
```

```dart
FlutterMap(
  options: const MapOptions(
    initialCenter: LatLng(31.2304, 121.4737),
    initialZoom: 12,
  ),
  children: [
    TileLayer(
      urlTemplate: 'https://tile.openstreetmap.org/{z}/{x}/{y}.png',
      userAgentPackageName: 'com.example.app',
    ),
    MarkerLayer(markers: [
      Marker(
        point: const LatLng(31.2304, 121.4737),
        child: const Icon(Icons.location_pin, color: Colors.red),
      ),
    ]),
  ],
)
```

**优点**:
- 全平台一致(Android / iOS / Web / 桌面)
- 免费(瓦片源根据使用量付费)
- 完全可定制

**缺点**:
- 需要自己接入瓦片服务
- 没有路线规划、POI 搜索等功能(自己接 API)

## 七、定位:geolocator(海外通用)

```yaml
dependencies:
  geolocator: ^13.0.1
  permission_handler: ^11.3.1
```

### 权限申请

iOS `Info.plist`:

```xml
<key>NSLocationWhenInUseUsageDescription</key>
<string>用于显示附近的店铺</string>
<key>NSLocationAlwaysAndWhenInUseUsageDescription</key>
<string>用于后台跟踪运动轨迹</string>
```

Android `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_BACKGROUND_LOCATION" />
```

### 用法

```dart
// 检查 / 请求权限
LocationPermission perm = await Geolocator.checkPermission();
if (perm == LocationPermission.denied) {
  perm = await Geolocator.requestPermission();
}

// 一次性
final pos = await Geolocator.getCurrentPosition(
  desiredAccuracy: LocationAccuracy.high,
);
print('${pos.latitude}, ${pos.longitude}');

// 持续
Geolocator.getPositionStream(
  locationSettings: const LocationSettings(
    accuracy: LocationAccuracy.high,
    distanceFilter: 10,    // 移动 10 米才回调
  ),
).listen((pos) => print('位置变化:$pos'));

// 距离计算
final distance = Geolocator.distanceBetween(31.2, 121.4, 39.9, 116.4);
```

⚠️ geolocator 拿到的是 **WGS84**,要在国内地图上显示先转 GCJ-02 / BD-09。

## 八、定位:amap_flutter_location(国内合规)

国内合规要求**位置 SDK 备案**。geolocator 没有合规备案,提交国内市场可能被拒。

```yaml
dependencies:
  amap_flutter_location: ^3.0.0
```

```dart
final loc = AMapFlutterLocation();

loc.setApiKey(iosKey: '...', androidKey: '...');
loc.setLocationOption(AMapLocationOption(
  desiredAccuracy: DesiredAccuracy.high,
  needAddress: true,
));

loc.onLocationChanged().listen((res) {
  print(res['latitude']);
  print(res['address']);    // 自动反查地址
});

loc.startLocation();
// loc.stopLocation();
```

高德返回 GCJ-02,显示在高德 / 腾讯地图直接用。

## 十三、隐私合规

国内 App 必须做:

1. **首次启动弹隐私协议**,用户同意才初始化定位 SDK
2. **后台定位**额外说明用途(送外卖、跑步、车辆调度)
3. **允许"仅使用期间"**而不是要 always
4. **用户可以一键关闭定位**

iOS 14+ 还区分"精确位置"和"模糊位置"。

```dart
final accuracy = await Geolocator.getLocationAccuracy();
if (accuracy == LocationAccuracyStatus.reduced) {
  // 用户只给了模糊位置,引导他开精确
  await Geolocator.requestTemporaryFullAccuracy(purposeKey: 'PreciseLocationUsage');
}
```

`Info.plist`:

```xml
<key>NSLocationTemporaryUsageDescriptionDictionary</key>
<dict>
  <key>PreciseLocationUsage</key>
  <string>用于精准导航</string>
</dict>
```

## 十四、性能 tips

1. **Marker 数 > 50 用聚合**,不然滑动卡
2. **大量 Polyline 简化**:`Douglas-Peucker` 算法去抽稀点
3. **缓存瓦片**:`flutter_map` 用 `flutter_map_tile_caching`
4. **地图组件懒加载**:进入页面才创建,离开 dispose
5. **小屏不要叠满 UI**:地图本身就该是主角

## 十五、常见坑

### 1. 坐标系混用

→ 显示偏移。**WGS84 / GCJ-02 / BD-09 转换三角永远要做**。

### 2. 地图卡顿

iOS 上 PlatformView 性能成本高(回顾 22)。**地图不要嵌在大滚动列表里**,放到独立页面。

### 3. 定位拿不到

- 权限没给
- 室内 GPS 弱(用网络定位 fallback)
- 高德定位:**忘记 `updatePrivacyAgree(true)`**(中国大陆合规)

### 4. Web 端用不了

`google_maps_flutter` Web 支持有限,有些功能(自定义 Marker)缺。
**Web 优先用 `flutter_map`**,跨平台一致性最好。

### 5. release 包高德 / 百度黑屏

→ keystore SHA1 没填到地图 console。每个 keystore 都要在地图后台备案。

### 6. Apple Maps 没插件

iOS 原生 MapKit 没有官方 Flutter plugin。要么用 google_maps_flutter,要么 PlatformView 自己包一层(回顾 16)。

## 十六、典型业务场景

### 1. 外卖 / 打车 — 实时位置

```dart
StreamSubscription _sub;

@override
void initState() {
  super.initState();
  _sub = Geolocator.getPositionStream(...).listen((pos) {
    _ctrl?.animateCamera(CameraUpdate.newLatLng(LatLng(pos.latitude, pos.longitude)));
  });
}

@override
void dispose() {
  _sub.cancel();
  _ctrl?.dispose();
  super.dispose();
}
```

### 2. 商家定位 — 附近店铺

调后端 API:`/shops?lat=&lng=&radius=` → 拿到店铺列表 → 渲染 Marker → 点击跳详情。

### 3. 路线导航

调路线规划 API → 拿到 polyline → 画线 → 配合 `Geolocator` 实时位置移动小车 icon。

## 十七、和已学知识的串联

- 定位结果转坐标系前要保证非 null,用 Riverpod AsyncValue 处理(09)
- 实时位置 stream 包成 StreamProvider(09)
- 地图页面进入 / 离开,Cubit init / dispose 释放资源(06 / 10)
- 距离计算放到 Isolate(18)避免阻塞 UI(几千个点排序)
- API key 用环境变量,生产 / 开发 key 区分(19 打包发布)

## 十九、心智模型

```
地图三件套:
  显示    → 地图 SDK(Google / 高德 / 百度 / OSM)
  定位    → Geolocator / 高德定位
  地理编码 → 地址 ↔ 坐标 → SDK API

国内合规:
  ✅ 隐私协议先弹
  ✅ 用户同意后初始化 SDK
  ✅ keystore SHA1 备案
  ✅ 坐标系不能混
```

地图开发的 90% 工作不是写 Flutter,而是**配置 SDK、坐标系处理、合规审核**。

入门后写起来其实很简单——所有地图的 API 都长得很像:**地图 + Marker + Polyline + Camera 控制**。理解了一个,其他都能秒上手。

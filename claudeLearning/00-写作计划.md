## 设计思路

四层结构,对应不同深度的需求:

```
工作流(用 Claude Code) → API 实战(用 Anthropic API)
   → SDK 与编排(写自己的 Agent) → 设计模式(成熟工程)
```

不重复 AI 系列已讲过的内容(Prompt Engineering、Tool Use 通用概念、MCP 协议本身),而是聚焦"在 Claude 这一家上怎么用、怎么落地"。

# 实战项目:管理台前端闭环

学完前端主线,至少要做一个真实项目。不要再做 Todo。

一句话先记住:**前端闭环项目 = 登录 + 权限 + 列表 + 表单 + 文件 + API 契约 + 测试 + 部署 + 监控。**

---

## 一、项目目标

做一个小型 SaaS 管理台:

```text
Acme Admin
  用户登录
  用户管理
  角色权限
  文件上传
  审计日志
  数据导出
  设置页
```

```text
React + Vite / Next.js
TypeScript
Tailwind + Radix / shadcn style
TanStack Query
React Hook Form + Zod

Sentry / Web Vitals
```

Vue 路线可以把 React 部分替换成:

```text
Vue + Nuxt
Pinia
VeeValidate / FormKit + Zod

```

## 四、路由

```tsx
const router = createBrowserRouter([
  {
    path: '/login',
    element: <LoginPage />,
  },
  {
    path: '/',
    element: <RequireAuth><AppLayout /></RequireAuth>,
    children: [
      { path: 'users', element: <UsersPage /> },
      { path: 'audit', element: <AuditPage /> },
      { path: 'settings', element: <SettingsPage /> },
    ],
  },
]);
```

鉴权边界:

```tsx
function RequireAuth({ children }: { children: React.ReactNode }) {
  const session = useSessionQuery();

  if (session.isLoading) return <PageLoading />;
  if (!session.data) return <Navigate to="/login" replace />;

  return children;
}
```

不要在每个页面重复判断登录。

## 五、API 契约

用户 schema:

```ts
export const userSchema = z.object({
  id: z.string(),
  name: z.string(),
  email: z.string().email(),
  role: z.enum(['owner', 'admin', 'member']),
  status: z.enum(['active', 'disabled']),
  createdAt: z.string(),
});

export type User = z.infer<typeof userSchema>;
```

列表响应:

```ts
export const userPageSchema = z.object({
  items: z.array(userSchema),
  total: z.number(),
  page: z.number(),
  pageSize: z.number(),
});
```

API client:

```ts
export async function listUsers(params: UserSearch) {
  const data = await request<unknown>(`/api/users?${toSearch(params)}`);
  return userPageSchema.parse(data);
}
```

不要等联调才发现字段错。

## 六、列表页

```tsx
export function UsersPage() {
  const search = useUserSearchParams();
  const users = useUsersQuery(search);
  const [selectedIds, setSelectedIds] = useState<string[]>([]);

  return (
```

## 八、权限

权限不要散落在按钮里:

```ts
export function can(action: Permission, user: SessionUser) {
  return user.permissions.includes(action);
}
```

```tsx
function Can({ action, children }: Props) {
  const session = useSession();
  if (!can(action, session.user)) return null;
  return children;
}
```

使用:

```tsx
<Can action="user.disable">
  <Button variant="danger">Disable</Button>
</Can>
```

注意:前端权限只负责隐藏 UI。服务端必须真正校验。

```ts

  await page.goto('/users');
  await page.getByRole('button', { name: 'Create' }).click();
```

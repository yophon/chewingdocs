е®һж—¶йҖҡдҝЎдёүеӨ§ж–№жЎҲ:

иҝҷдёҖзҜҮи®І SSE е’Ң WebSocket зҡ„е®һжҲҳдёҺйҖүеһӢ,йЎәдҫҝи®ІиҪ®иҜўзҡ„еҗҲйҖӮз”Ёжі•гҖӮ

1. еҝғжҷә вҖӢ

е®ўжҲ·з«ҜеҸ‘дёҖдёӘ HTTP GET иҜ·жұӮ,жңҚеҠЎз«Ҝ hold дҪҸиҝһжҺҘ,жҢҒз»ӯеҫҖйҮҢеҶҷж¶ҲжҒҜгҖӮ

зү№зӮ№:
  - еҚ•еҗ‘(жңҚеҠЎз«Ҝ вҶ’ е®ўжҲ·з«Ҝ)
  - еҹәдәҺ HTTP,жүҖжңүд»ЈзҗҶ / CDN / йҳІзҒ«еўҷйғҪж”ҜжҢҒ
  - иҮӘеҠЁйҮҚиҝһ
  - ж¶ҲжҒҜжңү id,ж–ӯдәҶйҮҚиҝһиғҪд»Һж–ӯзӮ№з»ӯ
  - ж ҮеҮҶжөҸи§ҲеҷЁ API(EventSource)

йҖӮеҗҲ:жңҚеҠЎз«ҜжҺЁйҖҒ(йҖҡзҹҘ / иЎҢжғ… / LLM жөҒејҸ)гҖӮдёҚйҖӮеҗҲ:е®ўжҲ·з«ҜиҰҒеҸ‘еӨ§йҮҸж¶ҲжҒҜеӣһжқҘ(йӮЈж—¶ WebSocket)гҖӮ

5. иҮӘеҠЁйҮҚиҝһ вҖӢ

ts
class ReconnectingWS {
  ws: WebSocket | null = null;
  private retries = 0;

  constructor(private url: string) { this.connect(); }

  private connect() {
    this.ws = new WebSocket(this.url);
    this.ws.onopen = () => { this.retries = 0; };
    this.ws.onclose = () => {
      const delay = Math.min(1000 * 2 ** this.retries, 30_000);
      this.retries++;
      setTimeout(() => this.connect(), delay);
    };
  }

  send(data: any) {
    if (this.ws?.readyState === WebSocket.OPEN) this.ws.send(JSON.stringify(data));
  }
}

WebSocket еҸӘжҳҜдј иҫ“еұӮ,ж¶ҲжҒҜж јејҸиҰҒиҮӘе·ұе®ҡгҖӮеёёи§Ғ:

7. жҲҝй—ҙ / йў‘йҒ“ вҖӢ

ts
const rooms = new Map<string, Set<WebSocket>>();

ws.on('message', (data) => {
  const msg = JSON.parse(data);
  if (msg.type === 'join') {
    const room = rooms.get(msg.room) ?? new Set();
    room.add(ws);
    rooms.set(msg.room, room);
  }
  if (msg.type === 'broadcast') {
    rooms.get(msg.room)?.forEach(c => {
      if (c !== ws && c.readyState === WebSocket.OPEN) c.send(JSON.stringify(msg));
    });
  }
});

ws.on('close', () => {
  rooms.forEach(r => r.delete(ws));
});

8. жЁӘеҗ‘жү©еұ• вҖӢ

е®ўжҲ·з«Ҝ A иҝһжңҚеҠЎеҷЁ 1,е®ўжҲ·з«Ҝ B иҝһжңҚеҠЎеҷЁ 2,A еҸ‘ж¶ҲжҒҜжҖҺд№ҲеҲ° B?

  жңҚеҠЎеҷЁ 1 ж”¶еҲ°ж¶ҲжҒҜ вҶ’ publish еҲ° Redis
  жүҖжңүжңҚеҠЎеҷЁ subscribe Redis,ж”¶еҲ°еҗҺиҪ¬еҸ‘з»ҷжң¬жңәзҡ„е®ўжҲ·з«Ҝ

import { Server } from 'socket.io';

  socket.on('chat', (msg) => {
    io.to(socket.handshake.query.room).emit('chat', msg);
  });

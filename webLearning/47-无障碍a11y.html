жң¬йЎөзӣ®еҪ•

ж— йҡңзўҚ a11y вҖӢ

a11y = "accessibility"(дёӯй—ҙ 11 дёӘеӯ—жҜҚз®Җз§°)гҖӮи®©ж®Ӣйҡңз”ЁжҲ·д№ҹиғҪз”ЁдҪ зҡ„зҪ‘з«ҷвҖ”вҖ”зӣІдәә / ејұи§Ҷ / иүІзӣІ / еҗ¬йҡң / иЎҢеҠЁдёҚдҫҝ / и®ӨзҹҘйҡңзўҚгҖӮ

дёәд»Җд№ҲйҮҚиҰҒ(дёҚеҸӘжҳҜйҒ“еҫ·):
  - 1 in 4 зҫҺеӣҪдәәжңүжҹҗз§Қж®Ӣйҡң(WHO)
  - жі•еҫӢиҰҒжұӮ(зҫҺеӣҪ ADA / ж¬§зӣҹ EAA / еӣҪеҶ…дҝЎжҒҜж— йҡңзўҚжқЎдҫӢ)
  - SEO еҠ еҲҶ(иҜӯд№үеҢ– HTML еҲ©дәҺзҲ¬иҷ«)
  - иҖҒе№ҙз”ЁжҲ· / дёҙж—¶йҡңзўҚ(жүӢйӘЁжҠҳ / ејәе…үдёӢ) д№ҹеҸ—зӣҠ
  - иӢ№жһң / и°·жӯҢ / ж”ҝеәңйҮҮиҙӯ ејәеҲ¶иҰҒжұӮ WCAG AA

дёҚеҒҡ a11y = жі•еҫӢйЈҺйҷ© + жөҒеӨұз”ЁжҲ· + жӢҝдёҚеҲ° B з«Ҝи®ўеҚ•гҖӮ

дёҖгҖҒWCAG:иЎҢдёҡж ҮеҮҶ вҖӢ

WCAG(Web Content Accessibility Guidelines)з”ұ W3C еҲ¶е®ҡ:

WCAG 2.1(2018)/ 2.2(2023)
еӣӣеӨ§еҺҹеҲҷ(POUR):
  Perceivable    еҸҜж„ҹзҹҘ вҖ” дҝЎжҒҜиҰҒиғҪиў«ж„ҹе®ҳиҺ·еҸ–
  Operable       еҸҜж“ҚдҪң вҖ” й”®зӣҳ / и§ҰжҺ§йғҪиғҪз”Ё
  Understandable еҸҜзҗҶи§Ј вҖ” еҶ…е®№жё…жҷ°
  Robust         еҒҘеЈ® вҖ” еҗ„з§Қиҫ…еҠ©жҠҖжңҜе…је®№

дёүдёӘзә§еҲ«:
  A    жңҖдҪҺ,еҹәжң¬еҸҜз”Ё
  AA   дё»жөҒзӣ®ж Ү,еӨ§йғЁеҲҶдјҒдёҡиҰҒжұӮ
  AAA  жңҖдёҘ,еҶ…е®№з«ҷзӮ№ / ж”ҝеәңжҺЁиҚҗ

зӣ®ж Ү:WCAG 2.1 AAгҖӮAAA е®һеңЁйҡҫиҫҫжҲҗ,A еӨӘдҪҺгҖӮ

дәҢгҖҒa11y еҸ—еҪұе“Қзҡ„з”ЁжҲ· вҖӢ

и§Ҷйҡң(зӣІ)
  вҶ’ еұҸе№•йҳ…иҜ»еҷЁ(VoiceOver / NVDA / JAWS / TalkBack)иҜ»еҮәжқҘ
  вҶ’ дҪ зҡ„йЎөйқўиҰҒ"иғҪиў«иҜ»еҮәжқҘ + йЎәеәҸеҗҲзҗҶ"

ејұи§Ҷ
  вҶ’ ж”ҫеӨ§еӯ—дҪ“ / й«ҳеҜ№жҜ”еәҰ
  вҶ’ еӯ—еҸ·иҰҒе“Қеә”зј©ж”ҫ + й…ҚиүІеҜ№жҜ”еӨҹ

иүІзӣІ(8% з”·жҖ§)
  вҶ’ дёҚиҰҒеҸӘз”ЁйўңиүІдј йҖ’дҝЎжҒҜ(зәўиүІй”ҷиҜҜ вҶ’ еҠ еӣҫж Ү / ж–Үеӯ—)

еҗ¬йҡң
  вҶ’ и§Ҷйў‘иҰҒеӯ—е№•,йҹійў‘иҰҒж–Үеӯ—зЁҝ

иЎҢеҠЁйҡңзўҚ / иҖҒе№ҙ
  вҶ’ й”®зӣҳд»Јжӣҝйј ж Ү / еӨ§зӮ№еҮ»еҢәеҹҹ / з®ҖеҚ•дәӨдә’

зҷ«з—«
  вҶ’ дёҚиҰҒжҜҸз§’й—ӘзғҒ > 3 ж¬Ў

и®ӨзҹҘйҡңзўҚ
  вҶ’ з®ҖеҚ•иҜӯиЁҖ,йҒҝе…Қжӯ§д№ү,жё…жҷ°зҡ„й”ҷиҜҜжҸҗзӨә

дёүгҖҒHTML иҜӯд№үеҢ–(еҹәзЎҖ) вҖӢ

жңҖеӨ§зҡ„ a11y жҸҗеҚҮжқҘиҮӘз”ЁеҜ№ HTML ж ҮзӯҫгҖӮеұҸе№•йҳ…иҜ»еҷЁйқ иҜӯд№үиҜҶеҲ«йЎөйқўз»“жһ„гҖӮ

еӣӣгҖҒARIA(еҝ…иҰҒж—¶еҶҚз”Ё) вҖӢ

1. ARIA жҳҜд»Җд№Ҳ вҖӢ

ARIA = "Accessible Rich Internet Applications" еұһжҖ§,з»ҷеұҸе№•йҳ…иҜ»еҷЁйўқеӨ–дҝЎжҒҜгҖӮ

html

ARIA з¬¬дёҖеҺҹеҲҷ:иғҪз”ЁеҺҹз”ҹ HTML е°ұеҲ«з”Ё ARIAгҖӮ<button> иҮӘеёҰжүҖжңү,<div role="button"> иҰҒеҶҷ 5 дёӘеұһжҖ§жүҚиғҪжЁЎжӢҹдёҖеҚҠгҖӮ

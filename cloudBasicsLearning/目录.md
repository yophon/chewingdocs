# 云服务与互联网基础常识

## 系列定位

这是一套面向网站、App、SaaS、独立开发和云部署的基础常识目录。

它回答的问题不是“某个按钮在哪里”,而是:

- 一个云服务免费套餐到底在限制什么
- DNS、CDN、SSL、WAF、DDoS、SLA 这些词分别处在哪一层
- Serverless、对象存储、托管数据库为什么便宜,又有什么边界
- 如何看懂请求数、流量、存储、构建分钟、日志保留、超额计费
- 小团队该怎么用云服务,避免一开始就过度设计或账单失控

## 大纲目录

### 第一层:互联网入口与访问链路

1. 云服务与互联网基础常识总览
2. 域名是什么:注册商、注册局、DNS 托管的区别

4. IP、端口与 HTTP:用户访问网站时发生了什么
5. HTTPS 与证书:SSL/TLS、CA、自动续期、证书错误

7. 反向代理与网关:Nginx、Cloudflare、API Gateway 的位置
8. WAF 与 DDoS 防护:它们能挡什么,挡不了什么

### 第二层:云资源与部署模型

9. 云服务器是什么:CPU、内存、磁盘、带宽、地域
10. 共享主机、VPS、云服务器、裸金属的区别
11. 容器与镜像:为什么部署不只是上传代码
12. Kubernetes 是什么:适合谁,不适合谁
13. Serverless 是什么:函数、冷启动、触发器、限制
14. 静态托管与前端部署:Pages、Vercel、Netlify、OSS 静态站
15. 对象存储:Bucket、Object、S3、R2、OSS、生命周期
16. 数据库服务:托管数据库、读写额度、连接数、备份
17. 队列、缓存与消息:Redis、MQ、异步任务的常识

### 第三层:稳定性、可观测性与恢复

18. SLA / SLO / SLI:服务承诺、内部目标、真实指标

21. 监控是什么:指标、日志、链路追踪的分工
22. 告警是什么:阈值、误报、漏报、升级路径

24. 备份、快照与恢复:备份不是有文件就行
25. 容灾与多地域:高可用、故障切换、成本边界

### 第四层:安全、身份与权限

26. 账号安全:Root 账号、MFA、子账号、最小权限

31. 防刷与 Bot 管理:验证码、限流、风控、误杀
32. 合规与数据位置:隐私政策、数据跨境、日志敏感信息
33. 安全责任共担模型:云厂商负责什么,你负责什么

### 第五层:计费、免费套餐与选型

41. Vercel / Netlify / Firebase / Supabase 免费套餐怎么看
